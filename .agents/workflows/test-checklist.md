# 测试验证清单

每次修改代码后，按以下清单验证关键功能：

## 基础流程
- [ ] 首页正常加载，"开始测试"按钮可点击
- [ ] 聊天对话正常进行，问题和选项正确显示
- [ ] 选项点击后页面自动滚动，问题可见
- [ ] 全部回答完成后显示"分析中"动画
- [ ] 结果页正常渲染，五行体质卡片正确

## 登录 / 未登录场景
- [ ] **未登录**：可正常做测试、查看结果
- [ ] **未登录**：右上角显示"登录"按钮（非"退出"）
- [ ] **未登录**：打卡区域不显示已打卡数据
- [ ] **未登录**：点击打卡时提示需要登录
- [ ] **已登录**：右上角显示用户名 + "退出"
- [ ] **已登录**：打卡数据正常读取和提交

## 结果页（index2.html 专有）
- [ ] 八卦圆盘可拖动旋转
- [ ] 点击扇形 → 圆盘旋转 → 内容在下方展开
- [ ] 再次点击同一扇形可收起
- [ ] 太极图清晰（黑白 S 曲线 + 鱼眼）
- [ ] 展开卡片文字清晰可读（浅色系）

## 通用功能
- [ ] 分享按钮正常工作
- [ ] 重新测试可以重新开始
- [ ] 页面在手机端响应式显示正常

## 双版本一致性
修改 `app.js` 时检查 `app2.js` 是否需要同步修改，反之亦然。
重点关注：`scrollToBottom`、打卡逻辑、认证逻辑。
