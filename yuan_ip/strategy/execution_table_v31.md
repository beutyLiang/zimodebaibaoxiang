## 一、合作模式

### 1.1 总体结构

```
M1 ──── M2 ──── M3 ──── M4 ──── M5 ──── M6 ──── M7+ ────→
│←── 代运营期（月费） ──→│←────── 分润期（15%收入） ──────→│
│   品牌方付：月费+硬成本  │  品牌方付：硬成本+15%收入分润   │
│   Zoey：全案操盘        │  Zoey：持续运营+享受增长红利    │
```

| 阶段 | 时间 | Zoey 收入方式 | 品牌方支出 |
|:---|:---|:---|:---|
| **代运营期** | M1-M3 | 固定月费 | 月费 + 硬成本 |
| **分润期** | M4 起 | 收入 x **15%** | 硬成本 + 收入15% |

### 1.2 代运营费定价

**品牌方总预算倒推法：**

| 项目 | 保守（15万） | 宽裕（20万） |
|:---|:---|:---|
| M1-M3 硬成本 | -15,000 | -15,000 |
| M4-M6 硬成本预留 | -15,000 | -20,000 |
| 预留缓冲 | -10,000 | -15,000 |
| **可用于代运营费** | **110,000** | **150,000** |
| **月均代运营费** | **~36,000** | **~50,000** |

**推荐定价：40,000 元/月（3个月合计 12万）**

| 维度 | 分析 |
|:---|:---|
| **品牌方总支出** | 12万(代运营) + 3万(硬成本M1-M3) + ~2万(M4-M6硬成本预留) = **约17万**，在15-20万区间内 |
| **市场定位** | 40,000/月 = 专业档中段，匹配"全案操盘"级别服务 |
| **品牌方心理** | "只付3个月，之后按效果分成"——进入成本可控，退出风险低 |
| **Zoey利润** | 月入40,000 - 团队成本8,000 = 月利润 **32,000**，3个月净赚 **~96,000** |

### 1.4 分润期规则（M4起）

| 条目 | 规则 |
|:---|:---|
| 分润基数 | 品牌全渠道营业收入（社群会费、卡牌、线下工作坊、企业服务等） |
| 分润比例 | **15%** |
| 结算周期 | 月度结算（次月15日前支付） |
| 最低保障 | 无（纯按效果） |
| 服务延续 | Zoey持续提供运营服务（内容+社群+AI维护），工作量随品牌增长调整 |

**Zoey 的长期回报测算：**

| 品牌年收入 | Zoey 年分润（15%） | 月均 |
|:---|:---|:---|
