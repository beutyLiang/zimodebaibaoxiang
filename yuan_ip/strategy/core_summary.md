## 三、平台矩阵——分层启动策略

| 层级 | 平台 | 启动时间 | 角色 |
|:---|:---|:---|:---|
| 🔥 **核心层** | **抖音 + 视频号** | M1 Day 1 | 流量引擎 + 私域沉淀 |
| 📚 **内容层** | **公众号** | M1 Day 1 | 深度内容中枢 + SEO |
| 📸 **辅助层** | **小红书** | M1 W2 起 | 结构图文 + 搜索 + 元气子策略 |
| 🌍 **国际层** | **YouTube + Substack** | M2 起 | 全球蓝海抢位 |
| ⚡ **国际短视频** | **TikTok** | M3 起 | 从 YouTube 素材剪辑 |

### 为什么分层而非一次铺开？

- **人力现实**：M1 只有 Zoey + 1 名兼职，无法同时运营 6 平台
- **策略聚焦**：先在国内跑通内容模型和变现逻辑，再国际化
- **成本控制**：国际线每月增加 2,000-3,000 元，放到 M2 更合理
- **数据验证**：M1 在国内验证爆款方向，M2 用验证过的模型做英文重构

### "一鱼多吃"内容分发

```
品牌方拍1条口播视频（3-5分钟深度版）
    │
    ├── 抖音：剪成 3条 15-60s 反常识金句
    ├── 视频号：1条 60-90s 精华版（含社交裂变CTA）
    ├── 公众号：1篇 深度长文（六步冲突法）
    ├── 小红书：2张 结构卡图文
    │
    └── [M2起]
        ├── YouTube：1条 3-5min 英文重构版
        ├── TikTok：2条 15-30s 英文金句版 [M3起]
        └── Substack：整合进周刊 Newsletter
```

## 五、拍摄外包方案

> 核心思路：**品牌方每周集中拍1次（2小时），一次拍完一周素材**

```
子默（周一）→ 输出 3 条脚本 + 1 篇长文 + 2 张图文
品牌方（周三）→ 集中拍摄 2h，产出 3-5 条素材
外包剪辑（周三-周五）→ 剪辑 3 条短视频
子默（周五）→ 审核 → 安排发布
```

| 推荐模式 | 月费 | 说明 |
|:---|:---|:---|
| **M1-M2：品牌方自拍 + 外包剪辑** | **~1,000-1,800元/月** | 性价比最优 |
| M3起可升级：摄影师上门 + 剪辑 | ~2,400-3,600元/月| 画面质感提升 |
| 品牌宣言片（仅 M1 一次性） | 5,000 元 | 专业团队拍摄 |
