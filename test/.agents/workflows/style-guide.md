# 初序 · 设计规范

## 品牌调性
- **关键词**：东方美学、自然节律、温暖内敛、专业可信
- **风格**：中式极简，不花哨但有质感

## 配色体系

### 五行主色
| 五行 | CSS 变量 | 色值 | 用途 |
|------|---------|------|------|
| 木 | `--wood` | `#3f8f78` | 主品牌色 |
| 火 | `--fire` | `#c75c5c` | 火行结果 |
| 土 | `--earth` | `#b8963e` | 土行结果 |
| 金 | `--metal` | `#8a8a8a` | 金行结果 |
| 水 | `--water` | `#4a7fb5` | 水行结果 |

### 基础色
| 用途 | 色值 | 说明 |
|------|------|------|
| 背景 | `#f5f0e6` | 宣纸色，温暖 |
| 文字主色 | `#1e2426` | 墨色 |
| 文字次要 | `rgba(30,36,38,0.75)` | 略透明墨色 |
| 边框 | `rgba(30,36,38,0.12)` | 淡灰 |
| 卡片底 | `rgba(245,240,230,0.88)` | 半透明宣纸 |

## 字体

```css
/* 标题/品牌 */
font-family: 'Noto Serif SC', serif;
/* 正文/UI */
font-family: 'Noto Sans SC', sans-serif;
```

## 设计原则

1. **浅色系为主** — 页面底色为宣纸色，不用深色/暗色背景
2. **文字必须高对比** — 深色文字在浅色背景上，永远不用白色文字
3. **圆角统一** — 卡片 `16px`，按钮 `14px`，小标签 `4px`
4. **动画克制** — 使用 `cubic-bezier(0.16,1,0.3,1)` 缓动，时长 0.3-0.5s
5. **间距呼吸感** — 卡片间距 `16px`，内边距 `14-18px`

## 文件结构
| 文件 | 职责 |
|------|------|
| `style.css` | 基础样式（两个页面共用） |
| `style2.css` | index2 八卦圆盘专用样式 |
| `app.js` | index.html 业务逻辑 |
| `app2.js` | index2.html 业务逻辑（含八卦圆盘） |

> ⚠️ 修改 `app.js` 中的通用逻辑时，记得检查 `app2.js` 是否需要同步！
