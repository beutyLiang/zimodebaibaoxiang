# Zoey 每日执行计划

> **品牌钩子：Stop overthinking. Make clear decisions.**
> **只做 3 件事：卖 Detox + 每天发内容 + 每周 1 新产品**

---

## ☀️ 上午（9:00-10:30）

### 9:00-9:30 — 小红书（初序 IP）
- [ ] 发今天的小红书帖子
- [ ] 回复昨天帖子的评论
- [ ] 看前天帖子数据，想想为什么好/不好

### 9:30-10:00 — 信息雷达（30 分钟）
- [ ] 10 分钟：Reddit 搜痛点（r/overthinking, r/productivity）
  - 搜："I struggle with" / "I feel stuck" / "can't decide"
  - 看到好的截图发子默
- [ ] 10 分钟：逛 Gumroad Discover + Product Hunt
  - 记下：产品名 + 定价 + 解决什么问题
- [ ] 10 分钟：X 看 Dan Koe / Sahil Bloom 最新推文
  - 回复 5 条优质评论（积累熟脸效应）

### 10:00-10:30 — 和子默对齐
- **问明哲：「明哲，给我今天的案例」**（每天固定，让明哲提供当天可用的营销/文案/策略案例）
- 告诉子默今天雷达看到了什么
- 子默汇报昨天数据（X/YouTube/Dodo）
- 确认今天要做的事

---

## 🌤️ 下午（自由安排）

### 产品工作（如果本周有新产品要做）
- 子默负责做，Zoey 审核
- 每周目标：1 个新的 $5-$9 产品

### 个人品牌内容（Build in Public）
- [ ] 写 1 条个人故事推文（创业过程/东方哲学感悟/真实经历）
- 不用包装，真实就行

### 每周 Newsletter（周三写）
- [ ] Substack 写 1 封邮件（干货 + 产品链接）
- 子默帮写初稿，Zoey 改语气

- 告诉子默今天执行情况
- 子默更新数据 + 准备明天内容

## 📅 每周团队节奏

### 周一 — 开周
- [ ] **问知远：** "上周数据怎么样？这周重点关注什么？"
- [ ] **告诉子默：** 这周要做什么产品 / 内容重点
- [ ] 子默排好本周每日任务

### 周二 — 正常执行
- 按每日计划走

### 周三 — Newsletter 日
- [ ] **子默** 写 Newsletter 初稿
- [ ] **发给明哲** 审英文
- [ ] 明哲改完 → Zoey 发 Substack

### 周四 — 正常执行
- 按每日计划走

### 周五 — 竞品日
- [ ] **问知远：** "帮我做一份本周竞品分析（Dan Koe / Sahil Bloom 发了什么）"
- [ ] 知远出报告 → 看有没有值得借鉴的

### 周六 — 正常执行 / 休息
- 按每日计划走（或休息）

### 周日 — 周报日（三人一起）
- [ ] **知远出数据：** 本周推文曝光 / 点击 / 订单 / 邮箱新增
- [ ] **明哲给方向：** 下周策略要不要调整
- [ ] **子默排任务：** 下周每日计划 + 新产品进度
- [ ] Zoey 确认下周方向

## 🎯 子默每天会提醒你的

每次新对话开始，子默会说：

1. **"小红书今天发第 X 天，内容是 XXX"**
2. **"30 分钟信息雷达做了吗？"**（可以问知远帮扫）
3. **"今晚推文和 YouTube 已准备好，20:00 来发"**
4. **"本周新产品进度：XXX"**
5. **周一额外提醒：** "该问知远要上周数据了"
6. **周三额外提醒：** "今天写 Newsletter，子默出初稿 → 明哲审"
7. **周五额外提醒：** "该问知远做竞品分析了"
8. **周日额外提醒：** "今天周报日，三人同步"

## 🔄 找谁做什么（速查）

| 情况 | 找谁 |
|------|------|
| 要做产品 / 写文案 / 干活 | **子默** |
| 要做决定 / 审文案 / 定价 | **明哲** |
| 要看数据 / 做调研 / 找方向 | **知远** |

---

## ❌ 不要做的事

- ❌ 花超过 30 分钟刷资讯（信息雷达有固定时间）
- ❌ 同时做 3 个以上产品（一次只做 1 个）
- ❌ 优化已有产品超过 1 小时（先卖，卖了再改）
- ❌ 没有数据就做决策（先问知远拉数据）
- ❌ 同一件事同时问三个人（知远调研→明哲判断→子默执行）
