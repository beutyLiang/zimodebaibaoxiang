## 🔥 一、与我们直接相关的三个重磅信号

### 信号 1：数字疗法纳入医保 — 健康AI商业化破冰

### 信号 2：豆包日均调用量突破 120 万亿 Tokens（两年增长 1000 倍）
- 火山引擎 4 月 2 日披露

- **豆包 AI 手机**（与中兴合作）预计 Q2 发布
- **对我们的意义**：我们在 Coze 上做 Bot 是正确选择。平台在快速增长，用户池在扩大

### 信号 3：小红书算法大改 — 评论权重 = 点赞的 4 倍
- 新 CES 评分公式：`点赞(1) + 收藏(1) + 评论(4) + 转发(4) + 关注(8)`
- 搜索流量占比超 65%，标题前 10 字必须包含核心关键词
- **AIGC 内容必须强制标注**，否则限流
- **对我们的意义**：
  - 我们的小红书内容策略要**重评论引导**（设计能引发讨论的话题）
  - 标题关键词要更精准（体质/养生/中医/调理）
  - AI 生成的图片/文案要标注"AI辅助创作"

## 📱 四、平台算法更新

### 抖音
| 变化 | 详情 |
|------|------|
| 权重模型 | 内容质量 35% + 用户行为 43% + 创作者权威 22% |
| 关键指标变化 | 完播率不再是唯一标准，**收藏/复访/追更** 权重更高 |
| GEO（生成引擎优化） | 能被 AI 搜索识别为"可信文档"的内容获得流量倾斜 |
| 搬运严控 | 上传素材打"暗水印"，抽帧剪辑搬运也会被拦截 |
| 垂直度要求 | 垂类内容占比建议 ≥ 85% |

### 小红书
| 变化 | 详情 |
|------|------|
| CES 评分 | 点赞(1) + 收藏(1) + 评论(4) + 转发(4) + 关注(8) |
| 搜索流量占比 | > 65%，标题前 10 字决定生死 |
| AIGC 规则 | AI 内容必须标注，否则限流 |
| KOS 模式 | 鼓励商家员工专业化分享 |
| 红线 | 禁止刷量、同 WiFi 多号互动、评论区过度引导私信 |
