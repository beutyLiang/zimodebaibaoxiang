## 二、用户画像

### 主力用户：25-35岁都市女性
- **痛点**：失眠、焦虑、月经不调、皮肤差、减肥反弹
- **需求**：不想去医院但想知道"我怎么了"、想要简单可执行的调理方案
- **场景**：午饭拍照问"能吃吗"、晚上睡不着找人聊、经期不舒服想知道怎么办
- **付费意愿**：免费功能 → 深度方案/产品推荐可付费

### 增长用户：宝妈群体
- **场景**：孩子发烧/过敏/挑食、自己产后恢复
- **特征**：高分享欲（妈妈群传播力极强）、高粘性

## 四、核心功能详细设计

### ⭐ 功能1：拍照问体质

**用户操作**：拍一张食物照片发给 Bot

**Bot 回复**（示例 - 水型用户）：
```
📸 识别到：冰美式咖啡

❌ 不太适合你

你是「深潜续航者」（水型体质），肾阳偏虚、怕冷。
冰咖啡性寒，会加重你的寒气，让你更容易腰酸和手脚凉。

✅ 建议替换：
→ 热拿铁（牛奶温脾，咖啡提阳）
→ 桂圆红枣茶（补肾阳首选）

💡 小贴士：水型的你，所有低于体温的饮品都要少碰。
```

**Coze 技术实现**：
- 使用 Coze 内置图片识别能力（多模态输入）
- 系统提示词中内置5种体质的食物适宜/禁忌知识库
- Bot 记住用户体质 → 每次拍照自动匹配

**为什么能赢**：
- "拍食物查热量"有 50 万用户，但只告诉你"多少卡"
- 我们告诉你"适不适合你" → **因人而异是中医的核心差异化**

---

### ⭐ 功能2：AI 舌诊

**用户操作**：伸舌头自拍发给 Bot

**Bot 回复**（示例）：
```
👅 舌诊分析完成

你的舌象：
• 舌体：偏胖大，有齿痕 → 提示脾气虚
• 舌色：偏淡 → 气血偏弱
• 舌苔：白腻 → 体内有湿气

📊 当前状态：你的土型特征加重了

上次（3月15日）你的舌象还算正常。
最近半个月是不是吃了很多生冷的？或者熬夜多了？

🌿 调理建议：
1. 本周少吃水果沙拉，换成温热的粥
2. 每天揉按足三里 3 分钟
3. 下周再拍一次舌头，我帮你对比变化

📸 保存这张分析卡，发给关心你的人看看 →
```

**Coze 技术实现**：
- Coze 多模态识别 + 系统提示词中内置舌诊知识
- 不做"医疗诊断"，做"体质状态追踪"（规避合规风险）
- Bot 记忆功能存储历次舌诊记录，做纵向对比

**为什么能爆**：
- **话题性极强**："我拍了舌头结果太准了！" → 小红书/朋友圈自传播
- **视觉冲击**：照片+分析结果 = 天然的传播素材
- **复购动力**：用户想看"调理后舌头有没有变化" → 每周回来

### 📋 功能3：体质测评（已有，优化）

**优化方向**：
- 原来5道题 → 保留，作为"没有舌诊照片时的快速通道"
- 测完结果升级：不只给体质名，给**7天入门调理方案**
- 加入人设名+金句（已定稿）：

| 体质 | 人设名 | 金句 |
|------|--------|------|
| 木型 | 静压指挥者 | 你看起来很稳，其实心里一直在打仗 |
| 火型 | 高能发光体 | 你一直在发光，但没人告诉你该停一下 |
| 土型 | 隐形支柱 | 你什么都扛，但身体已经在替你求救了 |
| 金型 | 边界守望者 | 你把一切都收得很紧，连呼吸都不放过 |
| 水型 | 深潜续航者 | 你不是不累，是已经习惯了带电量报警活着 |

---

### 🌿 功能4：节气养生推送

**逻辑**：24节气 × 5体质 = 120种个性化内容，自动轮转
- 每个节气（约15天）推送一次
- 内容包含：饮食建议、运动建议、作息调整、穴位推荐
- 用户进入 Bot 时自动触发："今天XX节气，你是X型体质，建议…"

---

### 🩸 功能5：经期+体质

**逻辑**：经期4阶段（经期→经后→排卵→经前）× 5体质
- 用户告知经期日期后，Bot 自动跟踪周期
- 每个阶段给出体质专属建议：
  - 土型经期："脾虚容易水肿，今天喝薏米水，别碰冰的"
  - 木型经前："肝气郁结容易暴躁，今晚泡澡加玫瑰花瓣"

---

### 🌙 功能6：睡前陪伴

**设计**：轻量级，不做完整冥想 App
- 用户说"睡不着"→ Bot 根据体质引导：
  - 水型："做3分钟腹式呼吸，我数给你：吸——2——3——4，呼——2——3——4——5——6"
  - 火型："你的心火太旺了，试试搓搓脚心的涌泉穴，搓到发热就行"
- 第二天早上可以回来说"昨晚睡得怎么样"→ Bot 记录睡眠质量趋势
