# Step 9：发布前最终检查 + 发布

## 发布前检查清单

- [ ] 系统提示词已粘贴且无遗漏
- [ ] 开场白已更新
- [ ] Bot 名称已改为"初序·体质管家"

## 发布步骤

1. 点击「发布」
2. 选择发布渠道：
   - ✅ 扣子商店（coze.cn）
   - ✅ 豆包 App
   - ✅ API（如果需要嵌入其他地方）
3. 等待审核通过（通常几分钟）
