# Step 3：插件与功能配置

## 操作步骤

### 3.1 开启图片理解能力
1. 在 Bot 编辑页面，找到「技能」区域
2. 确认模型选择为 **豆包 2.0 pro**（支持多模态/图片理解）
3. 如果有「图片理解」开关，打开它

> 💡 豆包 2.0 pro 本身就支持图片输入，用户可以直接发照片给Bot，不需要额外插件。

### 3.2 开启记忆功能
1. 在 Bot 编辑的「记忆」或「变量」区域
2. 开启「长期记忆」功能
3. 添加以下需要记忆的信息：

```
需要记住的用户信息：
- 用户的五行体质类型（木/火/土/金/水）
- 用户的人设名（如"静压指挥者"）
- 用户的经期日期（如果提供了）
- 用户之前的舌诊记录摘要
- 用户的特殊饮食偏好或过敏
```

### 3.3 插件配置（可选，增强版）
以下插件如果在技能商店找到就添加，找不到不影响基础功能：

| 插件名 | 用途 | 必要性 |
|--------|------|--------|
| 即梦/图片生成 | 生成体质结果卡片 | 可选（Phase 3再加） |
| ImageTool | 图片文字排版 | 可选（Phase 3再加） |

> **注意**：Phase 1 的核心功能（拍照问体质、舌诊、测评）都不需要额外插件。豆包 2.0 pro 的多模态能力已经够用。
