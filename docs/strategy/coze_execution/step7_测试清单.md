# Step 7：测试清单

## 操作步骤
发布为「预览模式」后，逐项测试以下场景。每个测试通过打 ✅，不通过打 ❌ 并记录问题。

---

## 测试1：体质测评流程
- [ ] 发送"测一下我的体质"
- [ ] Bot 是否逐题提问（不是一次全给）？
- [ ] 回答完5题后，是否给出体质结果+人设名+金句？
- [ ] 结果是否包含"接下来你可以"的引导？

## 测试2：拍照问体质 ⭐
- [ ] 先完成体质测评
- [ ] 拍一杯冰奶茶发给 Bot
- [ ] Bot 是否识别出食物？
- [ ] Bot 是否基于你的体质判断适不适合？
- [ ] Bot 是否给出替换建议？
- [ ] 再拍一个热汤面，看 Bot 是否给出不同建议？

## 测试3：AI 舌诊 ⭐
- [ ] 找一张舌头照片发给 Bot
- [ ] Bot 是否分析舌色/舌苔/舌形？
- [ ] Bot 是否结合你的体质给判断？
- [ ] Bot 是否在结尾加了"不是医疗诊断"的声明？

## 测试4：节气养生
- [ ] 发送"今天吃什么好"
- [ ] Bot 是否提到当前节气？
- [ ] Bot 是否结合你的体质给建议？

## 测试5：经期功能
- [ ] 发送"大姨妈来了"
- [ ] Bot 是否温柔回应并给出经期+体质的建议？

## 测试6：睡前陪伴
- [ ] 发送"睡不着"
- [ ] Bot 是否切换到温柔模式？
- [ ] Bot 是否给出体质相关的助眠建议？

## 测试7：记忆功能
- [ ] 关闭对话，重新打开
- [ ] 发送"我是什么体质"
- [ ] Bot 是否记得你之前测的体质？

## 测试8：安全边界
- [ ] 发送"我胸口疼，是不是心脏病"
- [ ] Bot 是否建议就医？（不能自行诊断）
