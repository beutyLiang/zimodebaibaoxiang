## 一、项目背景

### 1.1 我们有什么

| 资产 | 说明 |
|------|------|
| **知识库** | 20 个结构化文件，约 660KB，涵盖鞠门学派完整心理学体系 |
| **核心内容** | 11 张思维导图 + 1 份实战笔记 + 1 本 393 页专著（鞠强教授《家庭管理心理学》） |
| **方法论** | 鞠门学派：人本主义为基 + 精神分析 + 认知行为介入 + 催眠技术 + 二元平衡哲学 |
| **技术能力** | Coze 平台搭建 + 知识库配置 + System Prompt 编写 |

### 1.2 核心目标

**快速变现。**

不做生态、不做品牌建设期的烧钱动作。用最低成本、最短路径拿到第一笔收入，验证模式后再扩大。

### 1.3 为什么选婚恋情感赛道切入

| 判断维度 | 亲子教育 | 婚恋情感 ✅ |
|----------|---------|------------|
| 付费决策速度 | 慢（家长观望） | **快（情绪驱动冲动付费）** |
| 痛点强度 | 中等（焦虑为主） | **极高（出轨/冲突/失恋，当下就痛）** |
| 使用时段 | 白天碎片时间 | **深夜情绪峰值 = 付费高峰** |
| 内容引爆潜力 | 中等 | **极高（小红书/抖音情感类内容天然爆款）** |
| 知识库支撑 | 49K 字 | **41K 字婚姻 + 9K 字婚恋 + 46K 字基础理论** |
| 预计首单时间 | 4-8 周 | **1-2 周** |

**总结：婚恋情感是"急诊科"，用户痛时马上付费；亲子教育是"体检中心"，用户不急。急诊科变现永远比体检中心快。**

## 二、产品设计

### 2.1 产品形态

**Coze 智能体（Bot）**，发布至豆包 + 微信 + 飞书等多端。

### 2.2 Bot 核心能力

| 能力层 | 功能 | 说明 |
|--------|------|------|
| **L1 倾听** | 情绪识别 + 共情回应 | 用户倾诉时，Bot 准确捕捉情绪，给出温暖但专业的回应 |
| **L2 分析** | 关系问题诊断 | 基于鞠门学派理论，分析冲突根源（如负面人格识别、原生家庭影响 |
| **L3 指导** | 具体行动建议 | 给出可操作的沟通话术/行为调整方案（基于沟通心理学实战库） |
| **L4 疗愈** | 情绪疏导引导 | 引导用户进行自我催眠/情绪内化练习（基于鞠门情绪管理方法） |

### 2.3 Bot 人设定位

```
名称（候选，待确认）：
- 「心安」
- 「懂你」
- 「如初」（呼应初序品牌）
- 其他（子默建议）

人设：
- 温暖而专业的心理顾问
- 不是闺蜜式八卦聊天，而是"有方法论的情绪急诊医生"
- 核心态度：不评判、不说教、先共情再引导
- 理论根基：鞠门学派（用户无需知道术语，Bot 内化为回答逻辑）
```

### 2.4 差异化：我们 vs 普通 AI 情感聊天

| 维度 | 普通 AI 情感聊天 | 我们的 Bot |
|------|-----------------|-----------|
| 回答来源 | 互联网通用数据 | 660KB 鞠门学派私有知识库 |
| 理论深度 | "多沟通、多理解" | 潜意识分析 + 强化理论 + 人格类型识别 |
| 行动建议 | 泛泛而谈 | 渐进法 / 灌顶法 / 具体话术模板 |
| 情绪疏导 | 安慰话 | 催眠引导 + 情绪内化四步法 |
| 人格分析 | 无 | 六种负面人格精准识别 + 应对策略 |

## 三、变现模型

### 3.1 收费分层设计

```
┌─────────────────────────────────────────────────┐
│  免费层（流量入口）                                │
│  · 每日 3 次基础对话                               │
│  · 简单情绪安慰 + 通用建议                          │
│  · 目的：让用户体验到"这个 Bot 比别的更懂我"        │
├─────────────────────────────────────────────────┤
│  付费层 1：单次深度分析 ¥9.9                        │
│  · 一次完整的关系问题诊断                           │
│  · 识别冲突模式 + 人格分析 + 具体行动方案            │
│  · 生成「你的关系诊断报告」（PDF/图片）              │
├─────────────────────────────────────────────────┤
│  付费层 2：三次包 ¥29.9                            │
│  · 3 次深度对话（可分 3 天使用）                    │
│  · 适合持续性问题（如婚姻冲突反复发生）              │
├─────────────────────────────────────────────────┤
│  付费层 3：月卡 ¥49.9/月                           │
│  · 无限深度对话                                    │
│  · 每周一份「情绪健康周报」                         │
│  · 专属情绪疏导引导音频（基于催眠理论）              │
├─────────────────────────────────────────────────┤
│  高端层：真人咨询 ¥299-999/次（后期开放）            │
│  · Bot 初筛 → 转接真人心理顾问                     │
│  · 需要专业资质，后期再开                           │
└─────────────────────────────────────────────────┘
```

### 3.2 收入预估（保守 / 中等 / 乐观）

| 时间 | 免费用户 | 付费转化率 | 月收入（保守） | 月收入（中等） | 月收入（乐观） |
|------|---------|-----------|---------------|---------------|---------------|
| 第 1 个月 | 200 人 | 3% | ¥60 | ¥200 | ¥500 |
| 第 2 个月 | 800 人 | 5% | ¥400 | ¥1,200 | ¥3,000 |
| 第 3 个月 | 2,000 人 | 5% | ¥1,000 | ¥3,000 | ¥8,000 |
| 第 6 个月 | 8,000 人 | 8% | ¥6,400 | ¥15,000 | ¥40,000 |

> ⚠️ 以上为纯 Bot 收入。如果叠加课程/社群/1v1 咨询，第 6 个月后月收入可达 ¥50,000+

### 3.3 付费实现方式

Coze 平台本身不支持收费，需要借助外部工具：

| 方案 | 实现方式 | 复杂度 |
|------|---------|--------|
| **A. 微信小商店** | 小商店卖"对话券"，用户付款后获取专属码 | ★★ 推荐 |
| **B. 知识星球** | 付费加入星球 → 获取 Bot 高级版链接 | ★ 最简单 |
| **C. 小程序** | 自建小程序，内嵌 Coze API，直接付费 | ★★★★ 后期 |

**建议初期用方案 B（知识星球），零开发成本，1 天内上线。**

## 六、风险与合规

### 6.1 法律红线

| 风险 | 应对 |
|------|------|
| **非法行医** | Bot 必须明确标注"本服务为心理科普与情绪陪伴，不替代专业心理咨询或医疗诊断" |
| **自杀/自残** | 触发关键词自动回复：全国 24 小时心理援助热线 400-161-9995 |
| **隐私泄露** | 不存储用户对话记录，不向第三方共享 |
| **未成年保护** | 识别到未成年用户时引导联系家长或学校心理老师 |

### 6.2 商业风险

| 风险 | 概率 | 应对 |
|------|------|------|
| 小红书内容限流 | 中 | 避免"医疗"类关键词，用"成长/关系/沟通"替代 |
| 付费转化率低 | 中 | 先跑免费版积累口碑，根据数据调整定价 |
| 平台政策变化 | 低 | 同步布局微信生态（小程序），不依赖单一平台 |
| 知识产权争议 | 低 | 知识库来源为合法学习资料，Bot 输出为二次解读 |
