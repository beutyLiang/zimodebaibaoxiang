## 二、小红书算法深度分析

### 2.1 CES 评分公式（2026 版）

```
CES = 点赞×1 + 收藏×1 + 评论×4 + 转发×4 + 关注×8
```

**关键变化**：
- 评论权重 = 点赞的 **4 倍**
- 关注权重 = 点赞的 **8 倍**
- 长评论、深度阅读时长、截屏分享 → 额外加权
- 单纯点赞的价值大幅降低

### 2.2 流量分发链路

```
发布 → 初始池（200-500 曝光）
         ↓ 点击率 ≥ 2.5% + 互动率达标
       二级池（2000-5000 曝光）
         ↓ CES 分数持续增长
       三级池（1万+ 曝光）
         ↓ 搜索匹配 + 推荐双推
       爆款池（10万+ 曝光）
```

**前 1 小时是生死线** — 互动数据决定能否进入二级池

### 2.3 搜索流量 = 最大红利

- 搜索流量占比已达 **30%-65%**（取决于类目）
- 算法使用 BERT 模型分析语义，不再是简单的关键词堆砌
- **标题前 10 字**决定搜索命中率
- 搜推互通：搜索行为会影响推荐页 → SEO 做好 = 两个入口都吃到

### 2.4 健康养生赛道爆款公式

```
爆款 = 场景细化 + 真实痛点 + 明确方法论 + 视觉对比
```

**2026 年养生赛道 4 大趋势**：
1. 从"治病"→"日常养护"（治未病、季节调理）
2. "气血充盈 = 新美学"（养生与外在美挂钩）
3. 人群极度细分（备孕族/打工族/焦虑族/中年族）
4. 药食同源产品需求旺盛（黄精/五红粉/四神汤）

**爆款选题方向**：
| 类型 | 示例 |
|------|------|
| 食疗类 | "湿气重的人千万别吃这 3 种水果" |
| 体质类 | "你总觉得累？可能是脾虚，3 招自测" |
| 打卡类 | "21 天养脾胃计划，第 1 天从早餐开始" |
| 情绪类 | "焦虑失眠？试试这个中医安神方" |
| 对比类 | "坚持泡脚 30 天，我的舌苔变化（图）" |

### 2.5 红线警告

| ❌ 禁区 | 后果 |
|---------|------|
| 私信导流（"加微信""看主页"） | 限流/封号 |
| 刷量/同 WiFi 多号互动 | 永久降权 |
| AI 内容不标注 | 限流/标记低质 |
| 夸大医疗功效（"百分百有效"） | 笔记删除 + 账号处罚 |
| 用"治疗"而非"改善/调理" | 审核不通过 |

### 🎯 小红书优化建议

> [!IMPORTANT]
> **本周立即调整**

1. **标题公式升级**
   ```
   [人群词] + [痛点词] + [解决方案] + [数字钩子]
   ```
   **示例**：
   - ❌ "了解你的体质" ← 太泛
   - ✅ "久坐打工人总觉得累？3 招自测你是不是脾虚体质"
   - ✅ "湿气重的 5 个信号，第 3 个 90% 的人都中了"

2. **评论引导设计（每篇必做）**
   - 正文末尾加互动钩子：
     - "你是哪种体质？评论区告诉我 👇"
     - "你中了几个？评论区打 1234"
     - "想知道适合你体质的食谱吗？评论'食谱'我私发你"
   - **发布后 2 小时内回复前 20 条评论**（拉升 CES）

3. **搜索关键词矩阵**
   | 核心词 | 长尾词 |
   |--------|--------|
   | 体质 | 体质测评、体质类型、体质调理方法 |
   | 养生 | 春季养生、节气养生、养生食谱 |
   | 中医 | 中医养生、中医调理、中医食疗 |
   | 舌诊 | 舌苔发白、舌头有齿痕、AI看舌头 |
   | 湿气 | 湿气重怎么办、祛湿方法、湿气重的表现 |
   | 脾虚 | 脾虚怎么调理、脾胃弱、消化不好 |

4. **AIGC 合规**
   - 所有 AI 辅助生成的图片/文案 → 标注"AI 辅助创作"
   - 但要保持"真人感" → AI 出初稿 + 人工润色 + 真实经历植入

5. **内容节奏**
   - 稳定更新 > 偶尔爆款（算法奖励一致性）
   - 建议：**每周 3-4 篇**，固定时间发布
   - 每周复盘：看哪些笔记的"收藏率"和"评论率"高 → 复制模式

## 三、数字疗法 + AI 健康政策深度分析

### 3.1 政策里程碑

| 时间 | 事件 | 意义 |
|------|------|------|
| 2026 年初 | 《医疗服务价格立项指南》明确数字疗法收费 | 政策层面为 AI 健康"正名" |
| 2026 Q1 | 海南/北京/上海/广东试点医保报销 | **最高报销 70%** |
| 2026 年 | 10 万+ AI 医疗企业 | 行业已进入规模化阶段 |
| 2026 年 | AI 从"辅助"→"Agent"自主执行 | 技术成熟度跃升 |

### 3.2 用户接受度变化

```
2024: "AI 看病？不太靠谱吧..."
2025: "试试看，反正是辅助参考"
2026: "医保都报销了，应该靠谱"  ← 我们在这里
```

**关键转变**：
- 医保背书 = 官方认可 = 用户信任度大幅提升
- 慢病患者通过"数字处方"获得持续管理 → 用户已习惯 AI 健康服务
- "AI + 中医"被纳入"健康中国"国家战略支持

### 3.3 AI + 中医体质的市场机会

| 机会点 | 说明 |
|--------|------|
| AI 舌诊 | 技术已成熟（面部/舌头扫描 + AI 体质判读），消费者接受度高 |
| 个性化方案 | "千人千面"的体质管理 — 用户最核心的需求 |
| 药食同源电商 | 根据体质推荐 → 直接带货（黄精/枸杞/红枣等） |
| 慢病预防 | "治未病"是中医最核心的理念，与数字疗法方向完全一致 |
| 国标背书 | GB/T 46939-2025 中医体质分类标准 → 我们的测评有国家标准支撑 |

### 3.4 竞争壁垒分析

| 层级 | 我们的优势 | 我们的短板 |
|------|-----------|-----------|
| 知识壁垒 | ✅ 已有完整体质判读逻辑 + 6 大功能架构 | ❌ 缺少专业中医师背书 |
| 产品壁垒 | ✅ 全功能 Bot（测评+拍照+舌诊+节气+经期+陪伴） | ❌ 数据积累尚浅 |

### 🎯 数字疗法方向优化建议

> [!IMPORTANT]
> **中长期布局（本月启动）**

1. **强化"国标"背书**
   - 在 Bot 开场白和测评结果中加入：
     > "本测评基于《中医体质分类与判定》国家标准（GB/T 46939-2025）"
   - 在小红书内容中引用国标 → 增加权威性和搜索匹配

2. **建立"可信来源"身份**
   - 抖音 GEO（生成引擎优化）要求：能被 AI 搜索识别为"可信文档"的内容获得流量倾斜
   - 行动：写几篇**长文**（2000+ 字科普），发布在小红书/知乎
   - 目标：当用户问豆包/DeepSeek"哪个体质测评好"时，初序能被推荐

3. **合规红线清单**
   - ✅ 用"改善/调理/辅助" — 不用"治疗/治愈/疗效"
   - ✅ 每次输出加"非医疗诊断，如有不适请就医"
   - ✅ 小红书 AIGC 内容标注
   - ✅ 不做任何夸大宣传

4. **探索"数字处方"概念**
   - 虽然我们不是医疗机构，但可以借鉴"数字处方"的概念
   - 用户完成测评后 → 生成一份"体质调理方案"（类似处方的形式）
   - 视觉上模拟"处方单"设计 → 增加仪式感和信任感

5. **寻找中医师合作**
   - 找一位有资质的中医师做"首席内容顾问"
   - 不需要全职，只需定期审核 Bot 的回复逻辑
   - 可以在 Bot 中标注"内容经 XXX 中医师审核"
