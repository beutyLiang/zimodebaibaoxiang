## 五、X 平台深度运营（最被低估的平台）

### 为什么X最重要

```
TikTok = 泛流量，来得快走得快
X = 精准流量，一个真粉丝顶100个TikTok浏览
Substack 最大的流量来源就是 X
```

### X 每天必做的3件事

| 事项 | 怎么做 | 用时 |
|:---|:---|:---|
| **发原创帖** | 粘贴子默写的金句（每天2条） | 2分钟 |
| **互动回复** | 找3个philosophy/mindset大V的帖子→回复有深度的评论 | 10分钟 |
| **回复粉丝** | 每条评论都回复（子默帮写） | 5分钟 |

### X 增长技巧

```
1. 不要只发自己的内容，要参与别人的讨论
2. 回复大V时，发表有深度的观点（不是"Great post!"）
3. 用线程（Thread）写长内容 → 比单条帖子多5倍曝光
4. 每周发1个投票 → 算法会优先推荐互动高的帖子
```
