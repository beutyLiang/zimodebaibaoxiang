
**用「｜」分隔：前半段给AI看，后半段给人看。**

### 要建的合集

## 第三步：策划10个问答型视频（本周完成）

### 选题逻辑

```
去抖音搜索框输入关键词 → 看「下拉联想词」→ 那些就是用户真实在搜的问题
```

### 要做的操作

1. 打开抖音 → 搜索框输入以下关键词 → 截图保存联想词

| 搜索关键词 | 预期联想 |
|----------|---------|
| 失眠 | 失眠怎么办 / 失眠吃什么 / 失眠穴位 |
| 焦虑 | 焦虑怎么缓解 / 焦虑症状 / 焦虑自救 |
| 压力大 | 压力大怎么释放 / 压力大掉头发 / 压力大想哭 |
| 带娃累 | 带娃太累怎么办 / 带娃焦虑 |
| 情绪低落 | 情绪低落怎么调节 / 心情不好怎么办 |
| 湿气重 | 湿气重怎么调理 / 湿气重症状 |
| 脾虚 | 脾虚怎么调理 / 脾虚吃什么 |
| 气血不足 | 气血不足怎么补 / 气血不足的表现 |

2. 从联想词中选 **10 个**最匹配初序调性的问题
3. 每个问题做一个30秒视频 → 归入「身心自在」合集

### 10个视频模板（可直接用）

| # | 用户搜的问题 | 初序的回答 | 视频画面 |
|---|------------|----------|---------|
| 1 | 失眠怎么办 | 睡前3次深呼吸 + "不是为了入睡，是你值得好好休息" | 深夜枕边 + 蒸汽 |
| 2 | 焦虑怎么缓解 | 泡一杯姜枣茶的30秒 + "先给自己一个仪式感" | 茶杯特写 |
| 3 | 压力大想哭 | "哭不出来就看完这30秒" + 安静陪伴 | 窗边雨滴 |

| 6 | 湿气重怎么调理 | "不用急，先从一碗薏米水开始" | 碗里薏米水 |
| 7 | 气血不足吃什么 | "今晚多做一道你小时候喜欢吃的菜" | 厨房暖光 |

### 视频结构（固定模板）

```
0-3秒：标题文字浮现（= 用户搜的问题）
       例："失眠睡不着怎么办"
       
3-20秒：初序的回答（治愈文案 + 治愈画面）
        保持情绪陪伴感，不是教学
        
20-25秒：一句轻操作引导
          例："今晚试试：躺下，闭眼，深呼吸3次"
          
25-30秒：品牌卡
          「初序 · 回到最初的你」
```

**和纯情绪视频的唯一区别：开头3秒加了用户搜索问题作为标题。**

## 第四步：视频描述加结构化文字（每条+2分钟）

### 每条视频发布时，描述区写这个：

```
❓ [用户的问题]
💡 [初序的建议，1-2句]
🌿 想慢慢回来，留一个🌿

#搜索关键词1 #搜索关键词2 #情绪词 #初序
```

### 举例

```
❓ 失眠睡不着怎么办？
💡 不是教你方法，是提醒你：你值得好好休息。
   今晚试试躺下深呼吸3次，什么都不想。
🌿 想慢慢回来，留一个🌿

#失眠怎么办 #睡不着觉 #深夜治愈 #助眠 #初序
```

**为什么这样写**：豆包AI会读取视频描述文字来理解内容。结构化的❓💡格式让AI更容易匹配用户搜索问题。

## 第五步：粉丝过1000后开橱窗（到时再做）

### 操作步骤

1. 抖音 → 我 → 创作者服务中心 → 商品橱窗 → 申请开通
2. 绑定精选联盟（不需要自己进货）
3. 选品挂上：

| 优先选这些 | 客单价 | 对应视频 |
|-----------|--------|---------|
| 红枣姜茶 / 桂花茶 | ¥19-39 | 失眠/焦虑类视频 |
| 助眠薰衣草香薰 | ¥29-69 | 深夜治愈类视频 |
| 护手霜（文艺款） | ¥25-49 | "辛苦了"类视频 |
| 薏米/红豆粉 | ¥15-29 | 湿气/气血类视频 |

### 商品挂载对应关系

```
视频"失眠睡不着" → 挂「助眠花茶」商品链接
视频"湿气重"     → 挂「薏米红豆粉」商品链接
视频"辛苦了"     → 挂「护手霜」商品链接
```

> 不是每条视频都挂商品。🌙深夜治愈线**不挂**（纯情绪，挂商品会破坏信任）。🌿慢慢回来线**酌情挂**。
