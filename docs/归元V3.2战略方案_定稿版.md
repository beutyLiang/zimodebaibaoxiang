## 五、风险前置：按周迭代系统 ⭐

> **核心思维**：M1 不是"打基础月"，是"实弹测试场"。每周出数据、每周做决策，不等 M3 才发现问题。

### 5.1 M1 四周测试作战计划

#### W1（Day 1-7）：5发子弹测靶心

发布5条方向完全不同的内容，**刻意做差异化测试**：

| 编号 | 切口 | 形式 | 平台 |
|:---|:---|:---|:---|
| A | 内耗调节（无为） | 30秒口播金句 | 视频号 + TikTok英文版 |
| B | 决策焦虑（阴阳） | 图文信息卡 | 小红书 |
| C | 反向恐惧钩子 | 1分钟故事型视频 | 视频号+小红书 + TikTok |
| D | 热点结构解读 | 图文长笔记 | 小红书 + Substack英文版 |
| E | 互动测试型 | H5/投票 | 小红书+朋友圈 |

同步动作：注册全平台账号（国内6个+国际4个）、AI生成20个品牌视觉模板、建种子微信群。

**→ W1末 Checkpoint 0**：哪条播放最高？哪条互动率最高？**英文版 vs 中文版数据对比如何？**

#### W2（Day 8-14）：赢家翻倍，输家砍掉

- W1 数据最好的方向 → **同方向再做 5 条变体**（换标题/视觉/时长/发布时间）
- 表现最差的方向 → **直接砍掉**
- 英文内容保持节奏：TikTok 3条 + Substack 1篇 + X 每日金句

**→ W2末 Checkpoint 1**：

| 结果 | 判定 | 动作 |
|:---|:---|:---|
| 有1条 >1000 播放 | ✅ 苗头 | W3 全力加码此方向 |
| 全部 <500 播放 | 🟡 预警 | W3 执行"换弹药"（换人群定位/内容形式） |
| 全部 <200 播放 | 🔴 警报 | W3 立即启动"借势能"（博主互推+热点解读） |

#### W3（Day 15-21）：分支执行

**分支 A（有苗头）**：
- 赢家方向每天1条，连续7天
- 制作1个互动型测试（认知版本测试），引导分享裂变
- 首次直播 + 联系3-5个博主互推
- 1v1 认知咨询放出3-5个免费体验名额

**分支 B（无苗头）**：
- 停产同类内容，48小时内做三件事：
  1. 拆解10个同赛道爆款，逐条分析钩子/结构/视觉
  2. 换人群：从"认知升级"换到"职场内卷/创业焦虑/全职妈妈决策疲劳"
  3. 换形式：图文不行试口播，口播不行试对话/辩论体
- W3后半段：新方向再出5条测试

#### W4（Day 22-30）：锁定方向

| 月末状态 | 判定 | M2 策略 |
|:---|:---|:---|
| 有1条 >3000 播放 | ✅ Gate 0 通过 | M2 围绕此方向做饱和攻击 |
| 最好的在1000左右 | 🟡 继续迭代 | M2 继续A/B测试+启动博主互推 |
| 全部 <500 | 🔴 触发战略评估 | M2 第一周做战略诊断 |

同步完成：月度数据复盘、M2 排期、AI引擎 MVP prompt 架构规划、首期训练营大纲。
