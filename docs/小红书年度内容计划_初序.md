# 初序 · 小红书年度短视频发布计划（2026.04 — 2027.03）

## 🚨 合规红线（必须遵守）

### 绝对禁止使用的词
| 类型 | 禁用词 |
|------|--------|
| 疾病名称 | 便秘、哮喘、支气管炎、湿疹、痤疮、关节炎、高血压、糖尿病 |
| 功效承诺 | 治疗/治愈/根治/痊愈/消炎/杀菌/祛除/防癌/抗癌 |
| 绝对化用语 | 最/最佳/最有效/第一/顶级/极品/无敌/完美 |
| 效果保证 | 当天见效/XX天根好/一次性不复发/100%有效 |
| 药用暗示 | 药物/抗生素/激素/中药配方/医用级/药用 |
| 迷信用语 | 招财/化解小人/增加运气/开运 |

### 安全替代话术
| ❌ 不能说 | ✅ 可以说 |
|----------|----------|
| 治疗便秘 | 帮助肠道更舒服 |
| 降血压 | 支持心血管健康 |
| 减肥 | 体重管理 / 轻盈生活 |
| 增强免疫力 | 呵护身体防护力 |
| 排毒 | 帮助身体做减法 |
| 调理月经 | 关爱女性周期节律 |
| 祛湿 | 帮助身体轻松起来 |
| 这个方子包治 | 这个思路可以参考 |

### AI 内容规范
- AI 辅助创作的文字/视频必须标注「AI 辅助创作」
- 纯 AI 生成的画面需加专属标签
- 不标注 = 违规限流

### 引流规范
- 评论区放链接OK，私信引流高风险
- 不在笔记正文放微信号/二维码
- 链接放置顶评论，文案用"评论区有惊喜"引导

### 📹 视频 1：开篇立人设
- **标题**：我花了3年研究五行和肠道的关系，发现一个被忽略的真相
- **板块**：A 五行科普
- **形式**：✅ 真人出镜口播
- **发布时间**：周一 12:00
- **文案**：

> 你们有没有想过，为什么有的人怎么吃都不胖，有的人喝口水都长肉？
>
> 西医会告诉你是代谢问题，中医会告诉你是脾虚。
>

>
> 木行人天生代谢快但容易焦虑，火行人精力旺盛但肠胃敏感，土行人稳定但容易积滞……
>
> 你的体质类型，决定了你适合怎么吃、怎么睡、怎么调整情绪。
>
> 后面我会一个一个讲清楚，关注我，带你读懂你身体的说明书。

- **标签**：#五行体质 #中医养生 #肠道健康 #身体信号 #初序
- **拍摄指导**：
  - 场景：家里书桌/窗边，自然光，背景有绿植或书架
  - 穿着：简约质感，纯色上衣
