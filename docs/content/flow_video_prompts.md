# Yuanmind · Google Flow 视频生成指南 + 提示词

> **工具**：Google Flow（labs.google/flow）
> **模型**：Veo 3.1
> **画质**：最高4K
> **语言**：英文提示词效果最好

---

## 📱 使用步骤（第一次5分钟学会）

### 第一步：进入 Flow
```
1. 打开浏览器（需要VPN）
2. 访问 labs.google/flow
3. 用你的 Google 账号登录（Ultra套餐的那个）
4. 点 "New Project"（新建项目）
```

### 第二步：生成视频
```
1. 选择 "Text to Video"（文字生成视频）
2. 粘贴下面的提示词
3. 设置：
   - Aspect Ratio（比例）：9:16（竖屏，TikTok/Shorts用）
   - Duration（时长）：选最长
   - Quality（画质）：Standard（省积分）或 High（更好看）
4. 点 "Generate"（生成）
5. 等30-60秒 → 预览 → 满意就下载
```

### 第三步：发布
```
下载视频 → 打开TikTok APP → 上传 → 粘贴文案 → 发布
同一个视频 → 打开YouTube → 上传到 Shorts
```

## 💡 Flow 使用技巧

| 技巧 | 说明 |
|:---|:---|
| **不满意就重新生成** | 每次结果不同，多试几次 |
| **省积分** | 先用 Standard 画质测试，满意了再用 High 重新生成 |
| **调整风格** | 在提示词末尾加 "cinematic lighting" 或 "documentary style" 等 |
| **加中文元素** | 加 "Chinese calligraphy elements" 或 "ink wash painting style" |
| **竖屏很重要** | 一定要选 9:16，否则不适合 TikTok |

## 📊 积分预估（每月5000积分的情况下）

| 画质 | 每条消耗 | 你能做多少条 |
|:---|:---|:---|
| Standard | ~50积分 | ~100条/月 |
| High | ~105积分 | ~48条/月 |
| 4K | ~200积分 | ~25条/月 |

> **建议**：日常内容用 Standard，每周1条精品用 High。
