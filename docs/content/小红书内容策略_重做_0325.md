## 一、为什么牛油果和肚子胀有效？

### 数据回顾

| 帖子 | 观看 | 互动 | 选题类型 |
|------|------|------|----------|
| Day 1 牛油果争议 | **267** | 3新粉 | **你正在吃 + 你可能吃错了** |
| Day 2 腹胀 | **187** | 1赞1藏 | **你正在感受 + 原来不是我的错** |
| Day 8 八杯水 | 低 | 0 | 知识型 + 没痛感 |

### 核心发现：3 个共同点

**① 身体正在感受的痛（Physical Pain NOW）**
- 牛油果 → "我天天吃这个，难道吃错了？"
- 肚子胀 → "我现在就胀着呢！"
- 8杯水 → ❌ 没人正在因为喝水而难受

**② 打脸一个"你以为对"的东西（Belief Attack）**
- 牛油果 = "超级食物"→ 打脸 ✅
- 肚子胀 = 你以为是吃多了 → 其实是脾虚 ✅
- 8杯水 = 也是打脸，但 ❌ 痛感不够强

**③ "就是在说我"的瞬间（That's ME Moment）**
- 牛油果 → 女性都在吃，立刻代入
- 肚子胀 → 字面上的身体感受，不需要思考
- 8杯水 → ❌ 太理性了，需要"想一想"才能代入

### 一句话结论

> **有效的帖子 = 你身体正在经历的不适 + 你以为自己做对了但其实做错了**
>
> 无效的帖子 = 告诉你一个知识点（哪怕是反常识的知识点）

**区别在于**：一个让你**感受到痛**，一个让你**思考一个信息**。

小红书用户在刷手机，她不想思考，她想被**刺一下**。

## 三、新策略：从"讲知识"变成"刺痛人"

### 选题公式（新）

> **身体正在感受的不适 × 你以为自己做对了 × "原来是这个原因"**

### 选题必须过的 3 个关卡

| 关卡 | 检查问题 | 没过就砍掉 |
|------|---------|-----------|
| 1. 痛感 | 她看到标题时，身体会不会有反应？ | "8杯水" ❌ 没反应 |
| 2. 打脸 | 她会不会想"我一直以为这样是对的"？ | "下午困" ✅ "我以为是缺觉" |
| 3. 代入 | 她会不会觉得"这就是在说我"？ | "肚子胀" ✅ 正在胀的人立刻被锁住 |

### 封面规则（明哲定）

- **12 字以内**
- **必须"打脸 + 对立"**
- 不用漂亮图片，纯底色 + 大字就行
- 参考模板：
  - 你不是懒 / 是身体关机了
  - 越累越刷手机的人 / 都有这个问题
  - 你以为是拖延 / 其实是透支

### 正文必须有"锁人段"

在正文中间插入：

> 如果你：
> - （症状1 — 她正在经历的）
> - （症状2 — 她正在经历的）
> - （症状3 — 她正在经历的）
>
> 那你不是 XX，是 XX

这段决定她会不会点赞。

### 互动必须有低成本决策

结尾统一改成 A/B 选择题：

> 你更像哪种？
> A. XXX
> B. XXX
> 评论区打个 A / B

## 四、新选题清单（重做 7 天）

> 按痛感从强到弱排列，每个都过了 3 关卡检验

### Day 1（最强痛感，首发）

**选题**：一到晚上就不想动 + 明知该睡却停不下来
**封面**：你不是懒 / 是身体关机了
**打脸点**：你以为是自律差 → 其实是系统乱了
**锁人段**：一到晚上就不想动 / 明知道该睡却停不下来 / 白天又没力气
**互动**：A. 完全动不了 B. 白天累晚上清醒

### Day 2

**选题**：越努力养生身体越差
**封面**：很多人越养生 / 反而越虚
**打脸点**：你以为养生=做加法 → 其实你在透支
**锁人段**：买了很多保健品 / 跟着博主吃 / 身体反而更累
**互动**：A. 吃了一堆没用 B. 不敢再乱吃了

### Day 3

**选题**：戒糖30天皮肤反而暗沉（原 Day 11，好选题前移）
**封面**：我戒糖30天 / 皮肤更差了
**打脸点**：极端戒糖 ≠ 健康，甜味入脾
**锁人段**：脸色暗沉 / 精神变差 / 月经推迟
**互动**：A. 戒过糖没效果 B. 正在犹豫要不要戒

### Day 4

**选题**：下午犯困不是缺觉（原 Day 14，全民共鸣前移）
**封面**：一到下午就困 / 跟睡够没关系
**打脸点**：不是缺觉，是脾在报警
**锁人段**：吃完饭特别困 / 舌头有齿痕 / 容易腹胀
**互动**：A. 2点就困 B. 3-4点才困

### Day 5

**选题**：跑步伤膝盖（原 Day 12）
**封面**：每天跑步5公里 / 膝盖先废了
**打脸点**：不是所有体质都适合跑步
**锁人段**：跑完越来越累 / 膝盖下楼痛 / 越跑越没精神
**互动**：A. 跑完更累 B. 跑完精神好

### Day 6

**选题**：酸奶可能是糖水（原 Day 13）
**封面**：你喝的酸奶 / 就是一杯糖水
**打脸点**：一杯200ml酸奶 = 6块方糖
**锁人段**：每天喝酸奶"养生" / 以为0脂肪就健康 / 从来没看过配料表
**互动**：A. 从来没看过配料表 B. 看了吓一跳

### Day 7

**选题**：早睡反而失眠更严重（原 Day 9）
**封面**：我逼自己10点睡 / 失眠更严重了
**打脸点**：问题不在几点睡，在睡前状态
**锁人段**：躺下脑子还在转 / 越想睡越清醒 / 第二天更累
**互动**：A. 脑子关不掉 B. 身体累但清醒
