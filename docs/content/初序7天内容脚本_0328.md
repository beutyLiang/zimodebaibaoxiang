## 🔧 统一制作方式（全 AI，不拍摄）

### 步骤（每条视频 10-15 分钟）

```
① 打开即梦 AI → 输入画面提示词 → 生成 3-5 张图片     → 3 分钟
② 打开剪映 → 图文成片 → 导入图片                     → 2 分钟
③ 加文字（逐行淡入动效）                             → 3 分钟
④ 选 BGM（剪映自带治愈系音乐库）                      → 2 分钟
⑤ 导出 → 发布                                       → 2 分钟
──────────────────────────────────────────────────
总计：≈ 12 分钟/条
```

### 即梦 AI 画面风格统一提示词前缀

```
暖色调，柔光，浅景深，莫兰迪色系，安静氛围，
治愈系摄影风格，无人物面部，仅手部或物品特写，
4K，电影质感
```

| 项目 | 设置 |
