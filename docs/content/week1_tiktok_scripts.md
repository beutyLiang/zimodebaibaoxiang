# Yuanmind TikTok 文字动态视频 · 第1周脚本 + 制作教程

> **视频类型**：文字逐行出现 + 深色背景 + 安静音乐
> **时长**：15-30秒
> **工具**：剪映（CapCut）手机版
> **发布节奏**：每天或隔天1条

---

## 📱 制作教程（剪映/CapCut）

### 第一次做（5分钟学会，以后每条3分钟）

1. 打开**剪映**（国内）或 **CapCut**（海外版）
2. 点 **"新建项目"**
3. 选一张**纯黑色图片**作为背景（或深灰/深蓝）
   - 可以在手机上截一张纯黑屏保存到相册
   - 把时长拖到 **20-30秒**
4. 点底部 **"文字" → "新建文本"**
5. 输入第一句话 → 选白色字体 → 字号大一点
6. 调整文字出现的**时间点**（拖动时间轴）
7. 给文字加 **"淡入"动画**（点文字→动画→入场→淡入）
8. 重复步骤 4-7，逐句添加所有文字
9. 点 **"音频" → "音乐"** → 搜索 "meditation" 或 "zen" → 选一首安静的
10. 调低音乐音量到 **30-40%**
11. **导出** → 发布到 TikTok

### 小技巧
- 每句话之间停顿 **1-1.5秒**，让观众有时间读
- 最后一句停留 **3秒**，让人截图
- 视频比例选 **9:16**（竖屏）
- 最后加一行小字 **"@yuanmind1 · Follow for more"**
