# 初序 × 半山香学 · 借鉴融合方案

> 原则：**我们还是我们。** 松弛·东方·自愈的内核不变。
> 借鉴的是她的**结构、节奏和态度感**，不是她的暗黑风。

---

## 一、借什么 × 不借什么

| 借 ✅ | 不借 ❌ |
|-------|--------|
| **5 张图一组叙事**（不是 1 张封面+纯文字） | 暗黑色调（我们保持暖调） |
| **封面文字短而有力**（≤10 字） | 冷锋利的语气（我们保持柔和） |
| **两层文字系统**（大标题+小注解） | 纯态度输出（我们保持故事感） |
| **真实摄影质感**（有瑕疵的温度） | 无叙事的纯意境图（我们还是要讲故事） |
| **留白和呼吸感** | 纯黑背景 |

---

## 二、改版后的帖子结构（5 图制）

### 图 1 · 封面（Hook）
- **短句 ≤ 10 字**，要有"态度感"
- 半山用粗白字，我们用 **暖白粗宋体**
- 背景：暖色调暗调（深棕/墨绿/暖灰），不是纯黑
- 例：~~"后来我学会了不那么用力"~~ → **"不用力，也很好"**

### 图 2-3 · 内容图（故事）
- 真实场景照片 + **一两句走心短句叠加**
- 短句是正文里最戳心的那句话，不是完整段落
- 例：图上只写 **"风吹到脸上 / 突然觉得自己活着"**

### 图 4 · 金句图（收藏点）
- **两层文字**：大字态度 + 小字解释
- 这张图是被"收藏"的关键
- 例：
  - 大字：**"身体一直在等你"**
  - 小字：*不是等你更努力 / 是等你停下来*

### 图 5 · 品牌收尾
- 初序 logo 或品牌名
- 一句品牌 slogan：**"回到最初的节律"**
- 简洁干净，像杂志最后一页

## 三、Day 1 改版示例

**旧版**：1 张封面 + 长文案
**新版**：5 张图 + 短文案

| 图序 | 画面 | 图上文字 |
|------|------|---------|
| **图1（封面）** | 暖暗调窗台茶杯 | **"不用力，也很好"** |
| **图2** | 下班路上的树/光影 | "风吹到脸上的那一刻 / 是今天第一次觉得自己活着" |
| **图3** | 沙发一角/发呆的场景 | "五分钟。什么身份都不用扮演" |
| **图4（金句）** | 纯色暖调背景 | 大字：**"身体等了很久"** / 小字：*终于等到你不再催它了* |
| **图5（品牌）** | 极简干净底 | 初序 · 回到最初的节律 ☁️ |

**正文**（发在文字区，大幅缩短）：

> 睡了十个小时。醒来第一个念头——还能不能再睡一会儿。
>
> 不是困。是被抽空了。
>
> 后来只改了一件事：到家后，不立刻打开手机。坐着。五分钟。
>
> 那段怎么都睡不够的日子，就是这样过去的。
>
> 身体等了很久，终于等到你不再催它了。

**字数**：从 500+ 字 → **100 字以内**

## 五、色调升级

| | 旧 | 新 |
|---|---|---|
| 亮度 | 明亮暖调 | **降 2 档 → 暖暗调**（像傍晚的光，不是正午） |
| 饱和度 | 中等 | **再降 → 莫兰迪暗调** |
| 对比度 | 低 | **提高 → 让文字更突出** |
| 整体感觉 | 杂志内页 | **杂志封面** |
