## 一、《定位》—— 占领心智

### 核心理论
> 消费者的心里只有有限的"位置"。谁先占了，别人就进不去。

营销不是比谁的产品更好，是比**谁先在消费者脑子里占了那个词**。

### 最重要的 10 条商规

| 商规 | 原则 | 子默的理解 |
|:----:|------|----------|
| 1 | **领先定律** | 第一个进入心智的品牌赢。不是做得最好，是做得最早 |
| 2 | **品类定律** | 进不了第一？创造一个新品类让自己成为第一 |
| 3 | **心智定律** | 市场上不存在"最好的产品"，只存在消费者**认为**最好的 |
| 4 | **聚焦定律** | 什么都做 = 什么都不是。砍掉 80%，只做 20% |
| 5 | **专有定律** | 对手已经占了的词，你抢不走 |
| 6 | **对立定律** | 不是第一？把自己定位成第一的**反面** |
| 7 | **牺牲定律** | 想得到更多，必须先放弃一些东西 |
| 8 | **坦诚定律** | 承认缺点反而赢得信任（和医美方案里"劝你别做"一样！） |
| 9 | **延伸定律** | 品牌延伸 = 品牌稀释。做太多等于什么都不是 |
| 10 | **资源定律** | 再好的定位，没钱推也白搭 |

### 今天医美方案的反思
- 我前几轮的方案全部违反了"聚焦定律"——什么都想做
- "AI 效果验证平台"能拿到 60 分，因为它终于**聚焦**在一个点上
- **坦诚定律**和我们的"劝你别做"策略完全一致

## 二、《影响力》—— 6 个心理开关

### 消费者不是理性的。她被 6 个"自动开关"控制。

| 开关 | 原理 | 商业应用 |
|------|------|---------|
| **互惠** | 你给她好处，她觉得欠你的 | 免费 AI 皮肤检测 → 她觉得"应该在这消费" |
| **承诺一致** | 她说过的话、做过的选择，她会继续坚持 | 让她先填问卷/做测评 → 她已经"投入"了 |
| **社会认同** | 别人都在做，她也想做 | "80% 同龄人做过这个项目" → 她跟着做 |
| **权威** | 专家说的她信 | 医生 IP > 机构品牌 |
| **喜好** | 她喜欢你这个人，就更容易买你的东西 | 个人 IP 的本质就是"让人喜欢你" |
| **稀缺** | 越少越想要 | "本月仅剩 3 个名额" → 立即下单 |

### 组合应用案例

**医美 AI 平台的获客漏斗，用影响力 6 开关设计：**

```
第 1 步：免费 AI 皮肤检测（互惠 → 她觉得被给予了价值）
第 2 步：出报告，告诉她"你的状态排同龄人前 30%"（社会认同）
第 3 步：有的人被告知"你不需要做"（坦诚 → 信任感爆棚）
第 4 步：需要做的人看到医生 IP 讲解方案（权威）
第 5 步："本周预约有专属价"（稀缺）
第 6 步：做完后邀请她写评价（承诺一致 → 她公开说好，就会持续认同）
```

## 三、《疯传》—— STEPPS 框架

### 一个东西为什么会被传播？不是偶然，是因为它触发了这 6 个按钮。

| 按钮 | 含义 | 怎么用 |
|------|------|-------|
| **S - 社交货币** | 分享这个让我显得聪明/有品位 | "我做了很高级的 AI 皮肤检测" → 她愿意晒 |
| **T - 诱因** | 日常生活中的东西让她想起你 | 每次照镜子 → 想起"美丽时间线"App |
| **E - 情绪** | 引发强烈情绪的内容才会被分享 | 敬畏、愤怒、焦虑最强。"90%的人可能在做没用的项目" |
| **P - 公共性** | 看得见的东西才会被模仿 | 朋友圈晒"皮肤年龄降了 5 岁"的截图 → 朋友问"你用了什么" |
| **P - 实用价值** | 有用的东西人们愿意分享 | "这个 App 告诉我不用花 5000 块做光子" → 转发给闺蜜 |
| **S - 故事** | 信息包在故事里才能传开 | 不说"AI 检测"，说"有个女生花了 3 万做医美，后来发现只需要……" |

### 我们的内容创作检查清单

以后每条内容发布前，过一遍 STEPPS：
- [ ] 分享这条内容能让她显得聪明/有品位吗？（社交货币）
- [ ] 有日常诱因会让她想起这个内容吗？（诱因）
- [ ] 这条内容能激发强烈情绪吗？（情绪）
- [ ] 内容是否有"可见性"——朋友圈截图、分享卡片？（公共性）
- [ ] 内容对她有实际帮助吗？（实用价值）
- [ ] 内容是个好故事，还是干巴巴的信息？（故事）
