## 二、最重要的六个原则（商业应用深度解读）

### 1. 知己知彼，百战不殆 —— 用户画像 + 竞品分析

> **原文**：知彼知己者，百战不殆；不知彼而知己，一胜一负；不知彼，不知己，每战必殆。

**商业翻译**：
- **知己** = 我的产品是什么、我的优势在哪、我的弱点在哪
- **知彼** = 用户是谁、她怕什么、她什么时候掏钱；竞争对手在做什么
- **不知彼不知己** = 今天医美方案前 5 轮的我——既不了解行业，也没画用户画像

**教训**：这是今天被 Zoey 打回 8 轮的根本原因。以后每个方案，第一步"知己知彼"。

---

### 2. 不战而屈人之兵 —— 差异化 > 正面竞争

> **原文**：百战百胜，非善之善者也；不战而屈人之兵，善之善者也。

**商业翻译**：
- 打价格战打赢了 = "百战百胜"，但你累死了
- 做出差异化让对手没法跟 = "不战而屈人之兵"
- 案例：SBC 不跟小机构比价格，它做平台 → 小机构反过来加盟它

**应用**：在大健康方向，不要想"怎么比别人便宜"，要想"怎么让别人根本没法跟我比"。

---

### 3. 以正合，以奇胜 —— 基本面 + 意外感

> **原文**：凡战者，以正合，以奇胜。故善出奇者，无穷如天地，不竭如江海。

**商业翻译**：
- **正** = 产品质量、服务标准、合规运营（基本面）
- **奇** = 让客户意想不到的东西（创意+执行速度）
- 只有"正"= 平庸。只有"奇"= 不稳定。两者结合才能持续赢

**和 Zoey 三大壁垒的对应**：
- 正 = 认知 + 多维竞争力
- 奇 = 创意 + 执行速度

---

### 4. 避实击虚 —— 找空白市场

> **原文**：夫兵形象水，水之形，避高而趋下；兵之形，避实而击虚。

**商业翻译**：
- 不要往人多的地方挤（红海）
- 要像水一样流向低洼处（蓝海/空白市场）
- 案例：日本三宝乐啤酒作为市场第四，不在实体店竞争，转战电商 → 成功

**应用**：医美行业所有机构都在争"一线城市年轻女性"，谁去了"三四线"或"男性市场"谁就是避实击虚。

---

### 5. 先为不可胜 —— 先活下来

> **原文**：昔之善战者，先为不可胜，以待敌之可胜。不可胜在己，可胜在敌。

**商业翻译**：
- 先确保自己不会死（现金流、合规、基本客户量）
- 然后等对手犯错的时机
- 不要急着赢，先确保不输

**应用**：肠道菌群项目现在的状态就是"先为不可胜"——先跑通第一个客户，确保模式成立，再扩张。

---

### 6. 因粮于敌 —— 借力打力

> **原文**：智将务食于敌。食敌一钟，当吾二十钟。

**商业翻译**：
- 不要全部自己做，借别人的资源来壮大自己
- "因粮于敌" = 用竞争对手/合作伙伴的资源来降低自己的成本
- 案例：SBC 不自己开所有店，用加盟模式让别人出钱出力

**应用**：我们做肠道 AI 运营，不自己做检测，借检测公司的能力 + 我们的 AI → 共赢。

## 三、五事七计 —— 做任何决策前的检查清单

### 五事（评估是否该进入一个市场）

| 五事 | 含义 | 商业翻译 |
|------|------|---------|
| **道** | 上下同欲 | 团队是否对方向有共识？用户是否真的需要？ |
| **天** | 天时 | 时机对不对？政策支持吗？趋势在哪？ |
| **地** | 地利 | 市场环境适合吗？我在哪个城市做？ |
| **将** | 将帅 | 核心执行人有没有能力？ |
| **法** | 制度 | 流程标准化了吗？能复制吗？ |

### 七计（和竞争对手比较）

1. 主孰有道？→ 谁的团队更齐心？
2. 将孰有能？→ 谁的核心人才更强？
3. 天地孰得？→ 谁的时机和位置更好？
4. 法令孰行？→ 谁的执行力更强？
5. 兵众孰强？→ 谁的资源更多？
6. 士卒孰练？→ 谁的团队技能更好？
7. 赏罚孰明？→ 谁的激励机制更合理？

## 五、经典语录收藏

- "兵者，国之大事，死生之地，存亡之道，不可不察也" → 商业决策是生死大事，必须认真
- "夫未战而庙算胜者，得算多也" → 赢在准备，不是赢在战场
- "兵无常势，水无常形" → 没有永远正确的策略，只有永远灵活的思维
- "善战者，求之于势，不责于人" → 好的领导者创造有利形势，而不是怪团队不行
- "攻其无备，出其不意" → 在对手没防备的地方出手

## 六、智慧组合 —— 真实商战中的兵法组合拳

> Zoey 指示："每次执行商业计划时要学会智慧组合"
> 核心：兵法不是一条条用的，是组合在一起用的。

### 组合 1：华为 —— "先为不可胜" + "诡道" + "以正合以奇胜"

| 步骤 | 兵法原则 | 华为怎么做的 |
|------|---------|------------|
| 第 1 步 | 先为不可胜 | 20 年持续投研发，建起技术壁垒 |
| 第 2 步 | 诡道（藏而不露）| "备胎计划"——海思芯片藏了十几年不亮相 |
| 第 3 步 | 以奇胜 | 被制裁时一夜之间切换备胎，全世界震惊 |

**组合智慧**：先把底盘做到不可能输（正），同时秘密准备一个谁都不知道的杀手锏（奇），在关键时刻打出来。

### 组合 2：拼多多 —— "避实击虚" + "因粮于敌" + "用间"

| 步骤 | 兵法原则 | 拼多多怎么做的 |
|------|---------|-------------|
| 第 1 步 | 避实击虚 | 不跟淘宝/京东在一线城市打，从下沉市场切入 |
| 第 2 步 | 因粮于敌 | 借微信的流量（腾讯的资源）做社交裂变 |
| 第 3 步 | 用间（利用信息）| "砍一刀"让每个用户变成传播节点 |

**组合智慧**：去对手看不上的地方（虚），借别人的资源（粮），让用户自己帮你打仗（间）。

### 组合 3：Apple iPhone —— "不战而屈人之兵" + "攻其不备"

| 步骤 | 兵法原则 | Apple 怎么做的 |
|------|---------|-------------|
| 第 1 步 | 知己知彼 | 研究透了 Nokia/BlackBerry 的弱点（键盘+封闭系统） |
| 第 2 步 | 攻其不备 | 发明触摸屏——对手根本没想到手机可以没有键盘 |
| 第 3 步 | 不战而屈人之兵 | 创造了新品类，Nokia 的优势（键盘手感）变成了劣势 |

**组合智慧**：不在对手的战场上打（避开键盘手机比拼），创造一个全新战场让对手的优势失效。

### 组合 4：Tesla —— "兵贵神速" + "避实击虚" + "深入敌境"

| 步骤 | 兵法原则 | Tesla 怎么做的 |
|------|---------|-------------|
| 第 1 步 | 避实击虚 | 传统车企不做电动车 → Tesla 进入空白市场 |
| 第 2 步 | 兵贵神速 | 上海超级工厂从动工到投产只用 1 年 |
| 第 3 步 | 深入敌境 | 直接在中国（对手主场）建厂，切走份额 |

**组合智慧**：先找到没人去的地方（虚），然后以极快速度占领（速），再杀到对手家门口（深入）。

## 七、子默的"商业兵法"执行框架

> 以后每次做商业计划，按这个流程走：

### 出手前（庙算）

```
1. 知己 → 我们有什么？能做什么？弱点在哪？
2. 知彼 → 用户是谁？她几岁？她怕什么？她什么时候掏钱？
3. 五事检查 → 道（共识）天（时机）地（市场）将（人才）法（制度）
4. 七计对比 → 和竞争对手逐项比较，找到我强他弱的点
```

### 选择战场（避实击虚）

```
5. 红海在哪？→ 不去
6. 空白在哪？→ 去那里
7. 对手的弱点在哪？→ 攻那里
```

### 设计打法（组合出拳）

```
8. 正 = 基本面怎么做扎实？
9. 奇 = 什么是对手想不到的？
10. 借力 = 谁的资源可以借？（因粮于敌）
11. 速度 = 怎么比对手快？
```

### 执行中（灵活应变）

```
12. 兵无常势 → 随时调整，不要死守方案
13. 先为不可胜 → 每一步都确保自己不会死
14. 投之亡地然后生 → 必要时 all-in
```
