### 周日 3/16

**早餐 — 鸡蛋牛油果吐司**
```
食材：鸡蛋2个 + 牛油果半个 + 全麦面包1片 + 紫苏籽油半勺淋在牛油果上
做法：鸡蛋煎或水煮 → 牛油果切片放面包上 → 淋紫苏籽油 + 撒黑胡椒
搭配：1个橙子（维C促进胶原合成）
时间：8分钟
```

**运动 19:30 — 上半身力量（30分钟）**
```
热身：手臂画圈 前后各20次（2分钟)

动作1：哑铃弯举
→ 双手各握一个哑铃，弯曲手肘举到肩膀高度
→ 12次 × 3组，组间休息30秒

动作2：哑铃推举
→ 双手举哑铃到耳朵旁边，向上推到头顶
→ 10次 × 3组，组间休息30秒

动作3：跪姿俯卧撑
→ 膝盖跪地，手撑地面做俯卧撑
→ 8次 × 3组（做不了8个就做5个）

动作4：平板支撑
→ 手肘撑地，身体绷直
→ 坚持20秒 × 3组（坚持不了20秒就坚持10秒）

拉伸：手臂交叉胸前拉伸 每侧15秒（2分钟）
```

**晚上：炖银耳汤**
```
食材：干银耳1朵泡发 + 红枣5颗 + 枸杞一小把 + 冰糖少许
做法：银耳撕小块 → 放锅里加水 → 小火慢炖1.5小时 → 最后10分钟放红枣枸杞冰糖
→ 炖好后分成3份冰箱保存，周一周三周五各喝一碗
```

**睡前打卡：**
- [ ] 喝了8杯水
- [ ] 吃了补剂
- [ ] 完成运动
- [ ] 炖了银耳汤
- [ ] 22:30前上床
