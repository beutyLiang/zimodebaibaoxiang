## 🥗 饮食方案：抗炎抗糖化饮食

### 核心原则
1. **抗炎**：慢性炎症是衰老的第一推手
2. **抗糖化**：减少精制糖和高GI食物（糖化反应直接导致皮肤松弛和皱纹）
3. **高蛋白**：支撑胶原蛋白生成和肌肉维持

### 每日饮食模板

**早餐（7:15）选一种：**
- 🥣 燕麦粥 + 蓝莓 + 核桃 + 一勺蜂蜜
- 🥚 2个鸡蛋（水煮/煎） + 全麦面包 + 牛油果半个
- 🥤 蛋白质奶昔（牛奶/豆浆 + 1勺胶原蛋白粉 + 香蕉 + 少量坚果）

**午餐（12:00）核心结构：**
- ✅ 一巴掌大小的蛋白质（鱼/鸡胸/豆腐/虾）
- ✅ 两拳头的蔬菜（深色蔬菜为主：西兰花/菠菜/紫甘蓝）
- ✅ 一拳头的主食（糙米/红薯/荞麦面）
- ✅ 一勺好油（橄榄油/亚麻籽油拌菜）

**晚餐（18:30）轻量版：**
- 鱼类/豆腐 + 蔬菜沙拉/蒸菜 + 少量主食
- 晚餐分量 = 午餐的 **70%**
- ⚠️ 21:00之后不再进食

**零食（下午3点）：**
- 一小把杏仁/核桃（10颗左右）
- 或一个苹果 + 少量黑巧克力（70%以上可可）

### 重点多吃 ✅
| 食物 | 抗衰机制 |
|:---|:---|
| **三文鱼/深海鱼** | Omega-3抗炎，保护皮肤弹性 |
| **蓝莓/草莓** | 花青素，最强天然抗氧化剂之一 |
| **西兰花/菠菜** | 维C+叶酸+抗氧化物 |
| **牛油果** | 健康脂肪+维E，保护细胞膜 |
| **坚果（杏仁/核桃）** | 维E+好脂肪+蛋白质 |
| **绿茶** | EGCG是强效抗氧化和抗炎成分 |
| **骨汤/银耳汤** | 天然胶原蛋白来源 |

### 尽量少吃 ❌
| 食物 | 衰老机制 |
|:---|:---|
| 精制糖/甜食 | 糖化反应→皮肤松弛、暗沉 |
| 油炸食物 | 自由基→加速细胞氧化 |
| 过量酒精 | 脱水+炎症+干扰睡眠 |
| 过咸食物 | 水肿+血压升高 |
| 加工食品 | 反式脂肪+添加剂→慢性炎症 |

## 😴 睡眠方案：最强免费抗衰工具

> **科学事实**：长期睡眠不足（<6小时）会让皮肤衰老速度加快2倍，免疫力下降，皮质醇升高→加速全身衰老。

### 睡眠优化清单

- ✅ **固定时间睡觉**：每天 22:00-22:30 入睡（包括周末）
- ✅ **固定时间起床**：每天 7:00 起床（比睡够7小时更重要的是固定节律）
- ✅ **21:30后停用手机/电脑**：蓝光抑制褪黑素分泌
- ✅ **卧室全黑**：遮光窗帘，关掉所有指示灯
- ✅ **卧室温度**：18-20°C（凉爽环境促进深度睡眠）
- ✅ **睡前不喝咖啡/茶**：下午2点后不再摄入咖啡因
- ❌ **不要在床上工作/看手机**：让大脑把"床"和"睡觉"绑定

## 🧠 压力管理：皮质醇是衰老加速器

> **皮质醇**（压力激素）长期偏高会导致：皮肤变薄、胶原蛋白分解加速、腹部脂肪堆积、免疫力下降、睡眠质量恶化。

### 每日减压最小剂量（5分钟就够）

选一个每天做：
- 🫁 **呼吸练习**：4-7-8呼吸法（吸4秒→屏7秒→呼8秒）× 3次
- 📝 **感恩日记**：写3件今天值得感恩的事（研究证明可降低皮质醇23%）
- 🎵 **听5分钟音乐**：慢节奏音乐（<60 BPM）可生理性降低压力激素
- 🚶‍♀️ **5分钟户外散步**：自然光+绿色环境有即时减压效果
