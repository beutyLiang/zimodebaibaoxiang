### 🆕 调研 6：五行体质测评的法律定位——养生科普 vs 中医诊疗

> 来源：xinhuanet.com、dg.gov.cn、cctv.com、bucm.edu.cn、natcm.gov.cn、samr.gov.cn

**核心发现：五行体质测评属于"养生科普"范畴，普通人可以做，无需医疗资质。**

法律上存在**三个明确分层**：

| 层级 | 行为 | 谁能做 | 需要资质？ |
|------|------|--------|-----------|
| **科普层** | 讲解五行/体质知识、分享养生理念、推荐生活方式 | **任何人** | ❌ 不需要 |
| **辨识层** | 用标准化工具做体质辨识、给出养生建议 | 健康管理机构、养生从业者 | ⚠️ 推荐有，非强制 |
| **诊疗层** | 诊断疾病、开处方、做针灸/推拿治疗 | **只有执业医师+医疗机构** | ✅ 强制要求 |

**关键法规依据**：

1. **GB/T46939-2025《中医体质分类与判定》** — 2026年4月1日起实施
   - 它是**推荐性标准**（GB/T 中的 T = 推荐），不是强制标准
   - 适用范围明确写了："适用于医疗保健、公共卫生、科学研究、专业教学以及**健康产业**等全领域"
   - **非医疗机构（健康管理中心、体检机构等健康产业）也可以使用该标准**

2. **养生科普 vs 诊疗行为的法律界定**：
   - 养生科普 = 传播理念（饮食、起居、运动、情志调理）→ 任何人可做
   - 中医诊疗 = 诊断疾病 + 开处方 + 实施医疗技术 → 必须有执业资格

**初序的五行体质测评属于哪层？**

| 行为 | 所属层级 | 合规性 |
|------|---------|--------|
| AI对话问"最近睡眠怎样？容易发怒还是焦虑？" | 科普层 | ✅ 合规 |
| 根据回答告诉你"你偏向木/火/土/金/水型" | 科普/辨识层 | ✅ 合规（传统文化知识分享） |
| 推荐对应的生活方式建议（饮食、作息、运动） | 科普层 | ✅ 合规 |
| 推荐初序的对应疗愈视频合集 | 正常内容导流 | ✅ 合规 |
| ~~诊断"肝阳上亢"并推荐中药方~~ | ~~诊疗层~~ | ❌ 需要医资，不碰 |
| ~~声称"治疗/治愈"某疾病~~ | ~~诊疗层~~ | ❌ 违反广告法，不碰 |

**合规安全屏障**：在测评结尾加一句 —— "本测评仅供养生参考，不构成医学诊断建议。身体不适请就医。" — 这一句声明就把科普和诊疗划清了界限。

**核心原则：讲文化、讲养生、讲生活方式 = ✅。讲诊断、讲治疗、讲处方 = ❌。**

> **之前的判断修正**：我在初版评估中将五行检测直接等同于"中医诊疗行为"，要求医疗资质，属于过度保守。调研确认，作为养生科普工具定位完全合规。

## 三、需要修正的五个问题

### 修正 1：转化率假设偏高 3-5 倍 🔬

**有行业数据支撑。**

```
5万月曝光 → 10%关注(5000人) → ...    ← 子默假设
```

行业调研确认：**关注率均值 1-3%**，知识干货类上限3%。

### 修正 2：商业评估的"∞ ROI"是误导 📊

"资金投入为 0 所以 ROI 无穷大"是数学游戏。

```
正确的 ROI 计算：
ROI = 初序收入 / (Zoey时间 × 时间单价)

如果时间单价 = ¥30/小时：
  前3个月：ROI = ¥0 / (20h × ¥30) = -100%
  Month 4-6：ROI ≈ -50% ~ 持平
  Month 7+：ROI 开始转正
```

> ROI 不是 ∞。但边际时间成本确实很低（每条视频+5分钟），作为低成本实验仍然合理。

### 修正 3：抖音→微信导流存在严重合规风险 🚨

**经调研验证，风险比预想更高。**

```
子默方案：评论区留🌿 → 私信发微信号 → 微信私域
```

调研确认：
- 抖音 **D-Filter** 审核系统可实时识别微信号、谐音字、拼音首字母、图片二维码
- 处罚：轻则限流 → 重则**永久封号，申诉通过率极低**
- 评论区引导"看主页签名"也是**高发违规场景**

**修正后的合规链路**：

```
阶段1：粉丝群承接（合规 + 提升账号算法权重）
阶段2：企业号组件/小风车导企微（合规深度运营）
手册发放：通过飞书文档链接（抖音生态内合规）
```

### 修正 4：精选联盟AI推荐 — 取决于商品力而非账号大小 📦

**调研修正了我之前"对小号无效"的判断。**

豆包AI推荐排序逻辑：
- **商品力**（好评率、价格竞争力、转化率）决定排序
- **与达人账号大小无直接关系**

所以：初序挂的精选联盟养生茶能否被豆包推荐，取决于**这款茶本身的数据指标**。

但核心策略不变：**自有数字产品仍然是最大差异化**。精选联盟是锦上添花。
