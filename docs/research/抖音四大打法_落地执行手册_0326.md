## 打法 1：搜索 SEO 关键词矩阵

### 原理
> 不等算法推你，让用户搜到你。每条视频绑定一个长尾关键词，把抖音当百度来运营。

### 第一步：建立关键词库（本周完成）

**操作方法**：打开抖音搜索框，输入一级词，记录所有下拉建议词。

```
输入"湿气" → 记录：
  ├── 湿气重的人身体有什么表现
  ├── 湿气重怎么排出来
  ├── 湿气重吃什么好
  ├── 祛湿最有效的方法
  ├── 湿气重和脾虚有什么关系
  └── ...
```

**初序首批关键词库（50个起步）**：

| 一级词 | 长尾关键词（每个 = 一条视频选题） |
|-------|-------------------------------|
| **湿气** | 湿气重的表现 / 祛湿食疗方 / 湿气和脾虚的关系 / 红豆薏米水正确做法 / 湿气重能喝牛奶吗 |
| **体质** | 九种体质自测方法 / 阳虚体质怎么调 / 气虚和血虚区别 / 怎么判断自己体质 / 痰湿体质减肥 |
| **脾胃** | 脾胃虚弱的表现 / 健脾养胃吃什么 / 脾虚会导致什么 / 脾胃不好怎么调理 / 小孩脾虚怎么办 |
| **失眠** | 失眠中医怎么调 / 睡前喝什么助眠 / 失眠穴位按摩 / 肝火旺失眠 / 心脾两虚失眠 |
| **食疗** | 姜枣茶什么人不能喝 / 黑芝麻怎么吃最补 / 山药怎么吃健脾 / 枸杞泡水温度 / 桂圆红枣茶功效 |
| **节气** | 春分养生 / 清明养生 / 谷雨祛湿 / 立夏养心 / 小满吃什么 |
| **养生误区** | 泡脚的误区 / 喝水越多越好吗 / 晚上不能吃水果 / 生姜晚上能吃吗 |

### 第二步：关键词植入公式

每条视频的关键词必须出现在 **4 个位置**：

```
① 标题：「湿气重的人，身体会有这 3 个信号」
② 字幕/文字：视频画面中出现关键词
③ 口播/旁白：语音中说出关键词（算法会识别语音）
④ 描述+标签：#湿气 #祛湿 #中医养生 #体质调理
```

### 第三步：每周发布节奏

| 日 | 内容 | 绑定关键词 |
|---|------|----------|
| 周一 | 体质科普 | 体质类关键词 |
| 周三 | 食疗实操 | 食疗类关键词 |
| 周五 | 养生误区/冷知识 | 误区类关键词 |
| *跟节气* | 节气养生 | 节气关键词 |

**新手阶段每周 3 条**，质量优先。稳定后加到每天 1 条。

## 打法 2：IP 化（真实人设 > 专家人设）

### 原理
> 2026 最强 IP = "实战业务型 IP"。不是教别人，是**记录自己**。

### 初序在抖音的人设设计

| 维度 | 设计 |
|------|------|
| **身份** | 一个对养生感兴趣的普通人（不装专家） |
| **叙事视角** | 第一人称："我发现..."、"我试了..."、"我的身体告诉我..." |
| **差异化标签** | "用中医智慧管理自己身体的真实记录" |
| **调性** | 高级、克制、不卖焦虑（跟小红书统一） |
| **禁忌** | 不说"我教你"、不说"必须这样做"、不贩卖健康焦虑 |

### 人设落地：每条视频的叙事模板

**模板 A — 体感记录型（最推荐）**
```
钩子：「吃了一周姜枣茶，身体出现了 3 个变化」
正文：
  - 变化 1：早上起来不那么困了
  - 变化 2：手脚没那么凉
  - 变化 3：舌苔颜色变了
  - 中医怎么解释的：姜温胃散寒，枣补脾气
结尾：「你也可以试试，评论区告诉我你的感受」
```

**模板 B — 颠覆认知型（容易爆）**
```
钩子：「红豆薏米水祛湿？你可能一直喝错了」
正文：
  - 大部分人用的是圆薏米（寒性，越喝越湿）
  - 应该用炒薏米（温性，才能祛湿）
  - 加点芡实效果更好
结尾：「收藏起来下次泡对 ☕」
```

**模板 C — 节气指南型（长尾 SEO 强）**
```
钩子：「清明过后，别再喝红枣水了」
正文：
  - 清明后阳气上升，红枣补气助热
  - 这时候该换成菊花+枸杞（清肝明目）
  - 如果你是阳虚体质，红枣还可以继续
结尾：「你是什么体质？评论区回复 1 我告诉你该怎么调」
```

### 账号页面设计

```
昵称：初序 · 身体节律
头像：品牌 Logo（高级感）
简介：
  用中医智慧，读懂你的身体 🌿
  体质科普 | 节气养生 | 食疗记录
  回复「体质」领取九种体质自测表
背景图：统一品牌视觉
```

## 打法 3：深度内容系列化

### 原理
> 1 条 3 分钟深度视频 > 10 条 15 秒水视频。系列化内容让用户"追更"，铁粉率暴涨。

### 系列设计

#### 系列 1：「搞懂你的身体」（主力系列）

| 集数 | 标题 | 绑定关键词 | 时长 |
|:----:|------|----------|:----:|
| 01 | 你的身体一直在跟你说话，你听懂了吗 | 身体信号 | 2 min |
| 02 | 湿气重的人，身体会有这 3 个信号 | 湿气重表现 | 3 min |
| 03 | 脾虚 = 你身体的"发动机"坏了 | 脾虚表现 | 3 min |
| 04 | 为什么你总是累？气虚 vs 血虚一次搞懂 | 气虚血虚区别 | 3 min |
| 05 | 手脚凉不是体寒，可能是这个原因 | 阳虚体质 | 3 min |
| 06 | 舌头告诉你身体的秘密（附自测方法） | 舌诊自测 | 3 min |
| 07 | 九种体质，你是哪种？（完整版） | 九种体质自测 | 5 min |
| ... | 持续扩展 | ... | ... |

**统一封面模板**：系列名 + 集数 + 标题关键词，让主页看起来像一本"课程目录"。

#### 系列 2：「一周食疗实验」（真实感最强）

| 期数 | 实验内容 | 记录什么 |
|:----:|---------|---------|
| 01 | 一周姜枣茶 | 体温/精力/舌苔变化 |
| 02 | 一周红豆薏米 | 浮肿/排便/皮肤变化 |
| 03 | 一周早睡（22:30 前） | 第二天精力/情绪变化 |
| 04 | 一周不喝冷饮 | 脾胃感受变化 |

#### 系列 3：「节气养生日历」（全年 24 条，长尾价值极高）

每个节气出 1 条，标题格式统一：`「XX 节气」到了，身体需要你做这 3 件事`

这类内容**每年都有搜索量**，是真正的长尾资产。
