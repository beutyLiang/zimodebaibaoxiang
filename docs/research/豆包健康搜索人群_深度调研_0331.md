## 二、健康搜索人群规模（推算）

### 对标产品：蚂蚁阿福（唯一公开健康AI数据）

| 指标 | 数据 | 来源 |
|------|------|------|
| 总用户数 | **1 亿** ✅ | 证券时报、新华网 |
| MAU | **3000 万** ✅ | 央广网、虎嗅 |
| 日均健康咨询量 | **1000 万次/天** ✅ | 科技日报、证券时报 |

### 推算豆包健康搜索规模

```
逻辑：
- 豆包 MAU 3亿，蚂蚁阿福 MAU 3000万
- 豆包是通用AI助手，健康只是其中一个场景
- 行业数据显示健康是"仅次于办公/学习的第三大AI使用场景"
- 保守估计健康类搜索占豆包总使用量的 8-15%

推算结果：
  豆包健康相关 MAU ≈ 3亿 × 10% = 3000万
  豆包健康日均咨询量 ≈ 300万-1000万次
```

| 指标 | 推算值 | 置信度 |
|------|--------|--------|
| 豆包健康相关 MAU | **2400万-4500万** | ⚠️ 中（基于行业占比推算） |
| 日均健康咨询量 | **300万-1000万次** | ⚠️ 中 |

## 四、豆包给的解决方案（竞品分析）

### 豆包本身的回答方式

| 维度 | 豆包现状 | 评价 |
|------|---------|------|
| 回答形式 | 纯文字，结构化列点 | 🟡 正确但冰冷 |
| 有没有人设？ | 没有，通用AI口吻 | ❌ 没有温度 |
| 有没有后续？ | 没有，回答完就结束 | ❌ 用户看完就走 |
| 有没有推荐商品？ | 内测中，灰度测试 | 🟡 未全面上线 |
| 有没有陪伴感？ | 没有 | ❌ |

### 蚂蚁阿福的回答方式（最大竞品）

| 维度 | 蚂蚁阿福 | 评价 |
|------|---------|------|
| 回答形式 | 对话式 + 追问 + 生成报告 | ✅ 体验好 |
| 人设 | "健康伙伴"（偏理性） | 🟡 有人设但偏工具 |
| 后续 | 打通在线问诊、线下就医、智能设备 | ✅ 闭环完整 |
| 定位 | 医疗服务入口 | 偏"治病"而非"养生" |
| 用户群 | 三线以下城市55%，老年用户33% | 与初序画像不完全重叠 |

### Coze 上的养生 Bot（直接竞品）

| 竞品特征 | 详情 |
|---------|------|
| 已有赛道 | ✅ 五行体质辨识、中医养生已是热门类别 |
| 常见功能 | 体质问卷→五行匹配→食疗/穴位/运动建议 |
| 高阶功能 | 上传舌象照片（多模态）、生成养生图文 |
| 共同弱点 | **只给建议，没有温度。做完测评就走了，没有后续陪伴** |

## 五、用户画像

### 核心人群：3000万-4500万健康搜索用户画像

| 维度 | 画像 |
|------|------|
| **年龄** | **25-45岁** 为主力（占约60-70%） |
| **性别** | 女性占比略高（健康养生类搜索女性占约55-60%） |
| **城市** | 一二三线均有，但三线以下增长最快 |
| **角色** | 职场女性、全职妈妈、独居青年 |
| **收入** | 中等收入，愿意为健康付费但理性（100-500元舒适区） |

#### 三类细分人群

| 人群 | 年龄 | 核心痛点 | 搜索词 | 消费意愿 |
|------|------|---------|--------|---------|
| **A. 疲惫妈妈** | 30-45 | 带娃累、没人心疼、失眠、焦虑 | "带娃太累""失眠怎么办""气血不足" | ¥50-200/月 |
| **B. 职场打工人** | 25-35 | 内耗、颈椎、压力大、情绪低落 | "职场焦虑""久坐拉伸""心情不好" | ¥30-150/月 |
| **C. 养生觉醒族** | 28-40 | 主动健康管理、祛湿、食疗、体质调理 | "湿气重""八段锦""抗炎饮食""体质测试" | ¥100-500/月 |

#### 用户行为特征

| 行为 | 说明 |
|------|------|
| **"赛博树洞"心态** | 不好意思跟家人说的话，跟AI说 ✅ |
| **"刷→搜"闭环** | 刷抖音看到健康内容→跳到豆包搜更多→得到答案 ✅ |
| **深夜活跃** | 健康搜索高峰：22:00-01:00（失眠的人在搜失眠） ✅ |
| **结果型搜索** | 不搜"什么是失眠"，搜"失眠怎么办"→要的是解决方案 ✅ |
| **品牌信任度低** | 不信品牌广告，信"真实感受分享"和"像朋友一样的人" ✅ |

## 六、初序的截流机会

### 豆包给的 vs 初序能给的

| 她搜的问题 | 豆包给的答案 | 初序能给的 |
|-----------|------------|----------|
| "失眠怎么办" | 1.规律作息 2.减少咖啡因 3.就医 | "你不用那么努力入睡。今晚试试：躺下，闭眼，深呼吸3次。不是为了睡着，是你值得休息。" + 五行体质报告 |
| "压力大焦虑" | 1.运动 2.心理咨询 3.冥想 | "你已经撑了很久了。今天可以不坚强。" + 推荐初序的深夜疗愈视频 |
| "湿气重怎么调理" | 1.薏米 2.运动 3.就医 | "不用急，先从一碗薏米水开始。你的身体在等你慢下来。" + 对应体质的食疗方案 |

### 差距 = 机会

```
豆包回答：正确 ✅  有温度 ❌  有后续 ❌  有社区 ❌
初序回答：正确 ✅  有温度 ✅  有后续 ✅  有社区 ✅
                          ↑
                    这就是截流点
```
