轨道一

搞懂你的身体

引流新客户

👩

张姐

45岁 · 体检查出血糖 8.5

"回家百度越搜越怕，老公说'少吃糖'，闺蜜推荐的健康中心她不敢去。最后什么都没做，但那个 8.5 像石头压在心里。"

她缺的不是产品/方案/服务——她缺的是：有人用她听得懂的话告诉她，8.5 到底意味着什么。

1

引流 ·「一张图读懂你的体检报告」

在抖音/小红书/公众号发系列内容：

  • 📌 "血糖8.5到底严不严重？一张图告诉你"
  • 📌 "体检报告上这5个指标你一辈子不用管"
  • 📌 "医生说'定期复查'是什么意思？翻译给你听"
不是科普，是翻译。 把医学语言翻译成"你妈能听懂的话"。
2

第1次来 · 体检报告翻译

¥99 · 30分钟

把她报告上每一个她看不懂的数字，用大白话解释一遍：

  • "甘油三酯2.8 = 你的血液里油有点多了。不严重，但3-5年后可能变脂肪肝"
  • "甲状腺结节3mm = 90%的人都有，真的不用管"
  • "血糖8.5 = 这个要认真对待。其他的暂时不担心"
"你回去这周试一件事：晚饭7分饱，饭后散步20分钟。一个月后去社区卫生所测空腹血糖，拿结果来找我——免费看。"

她走的时候只拿走一张A4纸，什么都没买。但她得到了她这辈子在任何机构都没得到过的东西——有人坐下来把她的身体讲清楚了。

3

第2次来 · 她主动来的

她去社区卫生所测了血糖，拿结果回来找你。这次是她主动来的。

  • ✅ 降了："从8.5降到8.1了。你的身体是听话的，你对它好，它就在变好。"
  • ⚠️ 没降："没降不代表没用。你一般几点吃晚饭？我们看看哪里可以调。"
"接下来3个月我帮你管这个指标。每月去查一次，我帮你看变化、调方案。3个月 ¥597。3个月血糖没降，全退。"
4

自然复购 · 数据驱动

3个月后她的血糖从 8.5 降到 7.4。你不需要任何销售话术。

她续费是因为：数据在这里存了3个月 + 你是唯一把她身体讲清楚的人 + 血糖还没到7以下。

然后她会做一件你没让她做的事——告诉闺蜜：
"我终于搞懂我的体检报告了，那个人能把你的身体讲得特别清楚。"
轨道二

给你一个交代

激活私域老客户

👩‍🦰

王姐

48岁 · 去年消费 ¥8,200 · 现在不怎么来了

"花了那么多钱，也不知道到底有没有用。反正他们每次都说'还需要继续调理'。钱花了就花了吧。"

她的核心情绪不是不信任——是不甘心。花了钱 + 不确定有没有效 + 不敢承认被骗 = 沉默。

1

一条消息 · 破防

翻她以前的检测数据，发一条微信：

"王姐，我最近整理以前客户的健康档案，翻到你去年X月的记录。有个指标想提醒你——你当时空腹血糖7.9。建议你最近去社区卫生所查一下。不用来我们这里，社区就能查。就想提醒你一声。"

"不用来我们这里"这5个字 = 破防核心。因为她听过的所有话都是"你来你来你来"。

2

健康账本 · 复盘（免费）

她回来了。你做一件全行业没人做的事——给她看她的「健康账本」：

👤 王姐的健康账本（2024.3 - 2025.3）· 总消费 ¥8,200
✅
经络调理12次 → 肩颈疼痛评分从8→3 有效
¥3,600
❓
XX酵素产品 → 没有前后数据对比，不确定
¥1,800
⛔
XX排毒套餐 → 优先级不高，建议停掉
¥2,800

是的，你在告诉她"你以前花的一部分钱可能没必要"。

她会震惊。因为没有任何一家机构会主动说"这个钱你不用花"。

"从今天开始我们换个方式。你做任何项目之前，先测一个基线数据。做完再测一次。有没有效，数字说话。"
3

效果看得见 · 数据管理

3个月后翻开健康档案：

3月：肩颈疼痛 8分 → 经络调理 → 4月：5分 ✅ 有效
4月：睡眠质量 3分 → 芳疗+茶饮 → 5月：6分 ✅ 有效
5月：血脂 3.2 → 饮食调整 → 6月：2.8 ✅ 有效

她看到的不是"你又推了什么"——是"我的身体一步步在变好，每一步都有记录"。
