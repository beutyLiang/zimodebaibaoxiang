## 一、这本书讲什么？

一句话：**教你怎么用逻辑判断一件事到底对不对。**

殷海光是台湾著名哲学家，这本书用"老师和学生对话"的方式，把高深的逻辑学变成大白话。核心目的：让你不再被情绪、权威、模糊的话术忽悠。


## 三、核心工具（最重要的5个）

### 工具 1：分清"真假"和"对错"

```
真假 = 这个说法本身符不符合事实
对错 = 这个推理过程合不合逻辑

例子：
  "所有鱼都是哺乳动物"（假，但推理可以是对的）
  "所有哺乳动物都是温血的 → 狗是哺乳动物 → 所以狗是温血的"（推理对，内容也真）
```

> 🔑 **很多人吵架时混淆了这两件事。** 事实正确不代表推理正确，推理正确不代表结论一定是真的。

### 工具 2：三段论（最强的推理武器）

```
大前提：所有A都是B
小前提：C是A
结论：  所以C是B

例：
  大前提：所有SaaS产品都需要验证PMF
  小前提：AI社群助教是SaaS产品
  结论：  所以AI社群助教需要验证PMF ✅
```

**三段论的三大陷阱：**

| 陷阱 | 说明 | 例子 |
|------|------|------|
| **偷换概念** | 同一个词在前提和结论里意思不同 | "人民"在两个前提中定义不同 |
| **中项不周延** | 连接两个前提的中间概念不够广 | "有些A是B"不能推出"所有A是B" |
| **从否定推肯定** | 两个否定前提推不出肯定结论 | "不是所有X → 不是所有Y → 所以Z是Y"❌ |

### 工具 3：二难式（最常见的话术陷阱）

```
结构：
  如果你做A → 会有坏结果C
  如果你做B → 会有坏结果D
  你只能选A或B
  → 所以不管怎样你都完蛋

例：
  "你降价 → 利润没了 / 你不降价 → 客户跑了"
```

**破解二难式的三种方法：**

| 方法 | 怎么做 | 例子 |
|------|--------|------|
| **从角** | 证明还有其他选择（不止A和B） | "我不降价也不维持原价，我做套餐" |
| **冲角** | 反驳"如果A则C"这个因果关系 | "降价不一定利润没了，薄利多销啊" |
| **反二难** | 构造一个结论相反的新二难式 | 反过来论证"做A有好处C，做B也有好处D" |

### 工具 4：常见逻辑谬误（日常最实用）

| 谬误 | 大白话 | 日常例子 |
|------|--------|---------|
| **诉诸权威** | "X大佬说的所以一定对" | "马云说的所以一定对" ← 不一定 |
| **诉诸群众** | "大家都这么做所以一定对" | "10万人都买了所以产品好" ← 不一定 |
| **人身攻击** | 不反驳观点，反驳人 | "你又没创过业你懂什么" ← 跑题了 |
| **循环论证** | 用结论证明前提 | "这个产品好因为好评多，好评多因为产品好" |
| **以偏概全** | 用个案代替全部 | "我认识一个人做SaaS赚了所以SaaS好做" |
| **滑坡谬误** | 一步跳到极端结论 | "今天迟到明天就旷工后天就失业" |
| **稻草人** | 歪曲对方观点再反驳 | "你说要慢一点 → 你是说不干了？" |

### 工具 5：判别是非的三步检核

| 步骤 | 问自己 |
|------|--------|
| **① 厘清语意** | 他说的每个词，定义清楚了吗？有没有模糊/歧义？ |
| **② 分辨事实和情绪** | 这是在陈述事实，还是在表达情感？ |
| **③ 检验推理** | 他的推论形式对吗？前提能支持结论吗？ |

## 四、跟我们项目的关联

### 4.1 审方案时直接用

恺撒复审 AI 社群助教方案，我们可以用逻辑工具检验他的论证：

```
恺撒的论证：
  "B2B签单周期通常1-3个月 → 所以你8周签不到客户"

逻辑检验：
  大前提：B2B签单周期通常1-3个月（真吗？→ 通常是的 ✅）
  小前提：我们的目标是8周签客户（是的 ✅）
  结论：  8周签不到（严格推理 ✅）
  BUT → 中项"B2B"太宽泛。我们的第一个客户是熟人（肠道项目），
         不走标准B2B流程。所以这是个"中项不周延"的问题。

结论：恺撒的逻辑对一般情况成立，但对我们的特殊情况不完全适用。
```

### 4.2 写商业方案时避坑

| 我们容易犯的谬误 | 怎么避免 |
|---------------|---------|
| "Sahil Bloom 年赚 $10M → 我们也能" | ← 以偏概全。环境不同、资源不同 |
| "灵犀约指功能最强 → 所以能卖好" | ← 功能强≠卖得好。缺渠道缺声量 |
| "恺撒说的所以一定对" | ← 诉诸权威。他的保守估计也要检验 |
| "AI社群助教肯定行因为Sahil模式一样" | ← 类比不等于证明。要跑数据验证 |

### 4.3 跟客户谈判时用

二难式是最常用的谈判话术。学会识别和破解，谈判时不会被牵着走。

## 五、大白话总结

殷海光这本书就教了一件事：

> **别人说什么都别急着信，先拆开看看他的推理过程对不对。**

三个最实用的技能：
1. **听到"所以"两个字就警觉** — 前面的前提真的能推出后面的结论吗？
2. **听到"只有两个选择"就警觉** — 真的只有两条路吗？大概率还有第三条
3. **听到"大家都这么说"就警觉** — 大家都说不代表对

这本书不是教你赢辩论，是教你**不被忽悠**。
