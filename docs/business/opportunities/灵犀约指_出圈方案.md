## 二、用户使用：打造"成瘾闭环"

### 2.1 用户旅程设计

```
购买 → 首次使用 → 养成习惯 → 分享裂变 → 复购/升级

关键节点：
  ① 首次使用（决定留存的72小时）
  ② 第7天（形成习惯的关键期）
  ③ 第30天（产出体质档案的时刻）
  ④ 第90天（成为忠实用户/推荐者）
```

### 2.2 首次使用体验（决定生死的72小时）

| 时刻 | 用户看到 | 设计目标 |
|------|---------|---------|
| **开箱** | 精致包装+快速配对引导 | "比想象的好" |
| **首次佩戴5min** | 第一份实时心率+体温数据 | "哇，真的在测" |
| **首次睡眠后** | 完整睡眠报告+阴阳指数 | **"Wow moment"**（决定是否继续用） |
| **第3天早上** | AI推送第一条个性化养生建议 | "这个AI懂我" |
| **第7天** | 生成第一份周报+体质初步画像 | "想看接下来的变化" |

> 🔑 **核心指标**：首次使用后第7天留存率 ≥ 70%（低于此数据说明首次体验有严重问题）

### 2.3 长期活跃设计

| 功能 | 频率 | 目的 |
|------|------|------|
| **每日健康分** | 每天早上推送 | 养成"早上看一眼"的习惯 |
| **节气养生提醒** | 每15天 | 结合24节气推送调理建议（差异化） |
| **月度体质报告** | 每月1次 | "你的阴阳平衡这个月改善了12%" |

    └─────────────────┘           └──────────────────────┘

### 3.2 社群运营SOP

|--------|------|---------|---------|

### 3.3 私域变现模型：三层赚钱逻辑

```
戒指 = 入口（不靠它赚大钱）
服务 = 利润（持续收钱）
复购 = 增长（老用户再买）
```

#### 第一层：免费层（让人用起来）

买了戒指就**免费**用所有基础功能。目的不是赚钱，是**让用户养成每天打开App的习惯**。

```
用户心理：
  "¥1,200买了个戒指，App全免费，比Oura良心多了"
  → 好评 → 推荐朋友 → 口碑传播
```

包含：基础健康数据 + 阴阳指数 + 家庭空间

#### 第二层：付费层（持续收钱 ¥29-99/月）

用了1-2个月，用户已经离不开了，推出**会员服务**：

| 免费版能看到 | 付费版能看到 |
|-------------|------------|
| 阴阳指数（今天偏阳还是偏阴） | **AI帮你解读**："你最近阳虚，建议早睡+吃这3种食物" |
| 基础睡眠数据 | **深度体质报告**：你属于九种体质中的哪种，怎么调 |
| 家庭空间看数据 | **1对1 AI中医问诊**："最近总失眠，AI分析你可能是肝火旺" |

```
用户心理：
  "免费版只告诉我'阴阳失衡'，但不告诉我怎么办。
   花¥29/月就能让AI帮我出调理方案，值。"
```

**为什么投资人爱这个模型？**
- 每月收¥29-99 × 1万付费用户 = **¥29万-99万/月 持续收入**
- 不需要再生产硬件，纯软件利润 → **毛利 80%+**
- 用户越用越依赖（数据越积累越有价值）→ **流失率极低**

#### 第三层：硬件复购（放大收入）

用了18-24个月，出新一代戒指：

```
复购场景：
  → "灵犀2代出了，传感器更准了" → 自己换新
  → "妈妈用了我的，也想要一个" → 家庭套装（2只装）
  → "充电仓坏了/想换个颜色" → 配件复购
```

> 🔑 **一句话**：卖戒指是一次性的¥1,200，卖服务是每月¥29-99 × 一辈子。投资人看的不是"卖了多少戒指"，而是"有多少人每月付费"。Oura 估值 $52亿的原因——它不是硬件公司，是**健康订阅服务公司**。灵犀约指要走的就是这条路：**戒指是钩子，养生服务是生意。**
