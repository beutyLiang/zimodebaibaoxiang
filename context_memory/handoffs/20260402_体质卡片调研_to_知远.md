## 📋 需要你做什么

### 调研任务1：Coze平台图片生成能力
1. 扣子（coze.cn）的 Bot 是否支持在对话中**生成图片**？
2. 如果支持，用什么方式？（插件？工作流？API调用？）
3. 能否在 Bot 回复中直接展示生成的图片？
4. 图片质量如何？能否自定义模板/排版？

### 调研任务2：竞品体质/性格测评Bot分析
1. 在扣子商店/豆包搜索现有的"体质测评""MBTI""性格测试"类 Bot
2. 找出排名前3-5个，记录：
   - Bot 名称 + 链接
   - 用户量/热度
   - 测评流程（几道题、什么形式）
   - **结果展示方式**（纯文字？图片？卡片？）
   - 有没有分享/裂变机制
3. 总结它们做得好的地方和我们能借鉴的点

### 调研任务3：图片卡片替代方案
如果 Coze 不支持原生图片生成，调研以下替代方案的可行性：
1. 引导用户跳转到外部 H5 页面（我们自己做的）生成卡片
2. 使用 Coze 工作流调用第三方图片 API（如即梦API）
3. 其他市场上已验证的方案
