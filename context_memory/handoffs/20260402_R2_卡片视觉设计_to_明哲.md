## 📋 需要你做什么

为每种体质设计一张卡片的**视觉规范 brief**，包含：

1. **主色调**（1个主色+1个辅色，给出色号）
2. **视觉意象**（用什么元素代表这个体质？如：木型=被压住的火苗、水型=低电量图标）
3. **排版布局**（文字放哪里、人设名多大、金句多大、Logo位置）
4. **整体风格关键词**（3个词，如：侘寂/克制/暗光）

### 设计约束
- 风格统一：5张卡片必须是同一个系列感，不能各个风格不同
- 品牌色：初序的品牌调性是"东方极简+高级感+温暖"
- 底图尺寸：1080×1080px（小红书正方形最优）
- 文字区域必须预留足够空间（人设名+金句+初序Logo+二维码位）
- 参考风格：不要小清新，要**有分量感**——类似高端护肤品的海报调性

## 🚫 关键约束
- 不需要画图，只出文字 brief
- 每个体质的 brief 控制在5行以内
- 5张卡片的排版布局必须统一（只是配色和意象不同）
