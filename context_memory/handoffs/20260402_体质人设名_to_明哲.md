## 🎯 任务背景
我们的"初序"五行体质AI Bot（已上线扣子商店和豆包）目前测评完只给用户一段文字结果，没有分享冲动。为了让测评结果像MBTI一样可传播，需要给5种体质各设计一个有辨识度的"人设名"和一句扎心金句。

## ✅ 已完成的工作
- Bot 已上线，5道题测出用户五行体质（木/火/土/金/水）
- 每种体质有详细的中医特征描述（见下方参考）
- MBTI 的命名范式已验证有效（如"调停者""建筑师""指挥官"）

### 五种体质核心特征（参考用）

**木型** — 肝胆主导
- 容易焦虑、失眠、偏头痛
- 指甲脆、眼睛干涩
- 情绪压力大时身体先崩
- 核心矛盾：内心有火但一直压着

**火型** — 心/小肠主导
- 热情外向但容易透支
- 口腔溃疡、心烦、手心热
- 说话快、思维跳跃
- 核心矛盾：燃烧太快，阴被消耗

**土型** — 脾胃主导
- 容易胀气、消化不好
- 思虑过多、操心太多
- 嘴唇颜色淡，容易水肿
- 核心矛盾：扛起所有人但自己被掏空

**金型** — 肺/大肠主导
- 皮肤干燥、容易过敏
- 呼吸浅、容易感冒
- 完美主义、控制欲
- 核心矛盾：追求精确但身体在抗议

**水型** — 肾/膀胱主导
- 怕冷、腰酸、容易疲劳
- 总是硬撑、不知道累
- 性格沉稳但缺乏动力
- 核心矛盾：电池在告警但不肯停下来

## 📋 需要你做什么

1. 给每种体质设计一个**人设名**（2-4个中文字，像MBTI的"指挥官""调停者"风格，要有画面感、好记、容易传播）
2. 给每种体质写一句**金句**（15-25字之间，口语化，扎心但不伤人，像朋友说的话，不像医生说的话）
3. 每个体质出**3个备选方案**，标注你推荐哪个

## 🚫 关键约束
- 中文，口语化，不要学术味
- 不能有负面标签感（比如"虚弱者"❌）
- 金句要让用户看到就想截图发朋友圈
- 不超过500字总量

## 📦 交付物

格式如下，直接输出即可：

```
【木型】
方案A：人设名 — "金句"
方案B：人设名 — "金句"
方案C：人设名 — "金句" ⭐推荐

【火型】
...（同上格式）
```
